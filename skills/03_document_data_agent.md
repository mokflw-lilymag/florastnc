# 안티그래비티 스킬: [문서 및 데이터] "PDF랑 엑셀, 한 치의 오차도 없이"

## 사용 목적
`pdf-lib`, `html-to-image`, `xlsx` 등 폰트나 인코딩, 레이아웃 이슈가 복잡한 문서 조작/생성 작업을 처리할 때 사용합니다.

## 시스템 프롬프트
"**pdf-lib**과 **@pdf-lib/fontkit**을 사용해서 한글 폰트가 임베딩된 커스텀 영수증 PDF를 생성해야 해. **html-to-image**로 캡처한 DOM 이미지와 **Supabase**에서 가져온 결제 데이터를 조합해서 PDF 레이아웃을 잡는 함수를 작성해줘. 폰트 로딩 실패나 메모리 부족 상황에 대한 방어 코드도 잊지 마!"

## 핵심 가이드라인
* 한글 폰트 임베딩 등 로컬/웹 환경에서의 깨짐 이슈를 선제적으로 방어.
* 고해상도 이미지를 PDF화 할 때 발생하는 브라우저 메모리 부하 방어 로직 제안.
* 치명적 크래시를 막기 위한 `try...catch` 및 Fallback 전략 필히 포함.
