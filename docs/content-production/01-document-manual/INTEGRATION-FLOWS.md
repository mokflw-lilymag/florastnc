사장님이 **「저장 누르면 뭐가 이어지지?」**를 한눈에 보는 문서입니다. Track M(버튼 백과) 각 화면 끝 **「연결」** 절과 상호 링크합니다.

## 0. 읽는 순서 (매뉴얼 구조)

1. **1장 환경 설정** — 규칙·프린터·배송비를 **먼저** 잡는다 (`#settings-map`)
2. **연동 한눈에** — 저장 시 자동으로 이어지는 것 (`#integrations-hub`)
3. **상황별 가이드** — 급할 때 순서만 (`#sit-hub`)
4. **일상 메뉴 백과** — 버튼 단위 (`#nav-*`)
5. **PR 리본 백과** — 리본만 버튼이 많아 **별도 작성 단계** (`#nav-printer` …)

## 1. 주문 저장 → POS 영수증·주문서·예약증 (자동 인쇄)

**전제:** 환경 설정 → **프린터/브릿지**에서 영수증 브릿지(또는 윈도우 앱)가 **연결됨** 상태.  
**매장 PC**에서 브릿지/앱이 켜져 있어야 폰·다른 PC에서 넣은 주문도 인쇄됩니다.

### 1.1 수령 방식별 — 저장 직후 자동 출력

| 새 주문에서 고른 수령 방식 | 앱 표기 | 저장 직후 POS 자동 인쇄 | 비고 |
|---------------------------|---------|-------------------------|------|
| 매장 수령 | `store_pickup` | **없음** | 현장에서 바로 드리는 주문 — 자동 인쇄 생략 (재출력은 가능) |
| 수령 예약 (픽업 예약) | `pickup_reservation` | **예약증** 1장 (`pickup_shop`) | 픽업 일시·수령인 확인용 |
| 배송 예약 | `delivery_reservation` | **주문서** + **매장 영수증** (`order_form` + `delivery_shop`) | 기사용 **인수증**은 자동이 아님 → 필요 시 주문 목록에서 재출력 |

**한 줄 요약:**  
> **픽업 예약** = 예약증만 자동 · **배송 예약** = 주문서+영수증 자동 · **매장 수령** = 자동 인쇄 없음

### 1.2 수동 재출력 (주문 현황)

주문 목록 행 메뉴(⋮) 또는 상세에서 **인쇄** 관련 항목으로 다시 뽑을 수 있습니다.

| 수령 방식 | 재출력으로 뽑을 수 있는 것 |
|-----------|---------------------------|
| 픽업 예약 | 주문서 / 예약증 |
| 배송 예약 | 주문서 / **인수증**(기사용) / (별도) 매장 영수증 |
| 매장 수령 | 주문서 / 예약증 형식 |

매뉴얼 앵커: `#nav-orders-row-menu` · `#integration-print-reprint`

### 1.3 리본은 POS 자동 인쇄와 **별개**

- 경조사 **리본**은 영수증 프린터가 아니라 **리본 프린터 메뉴** 또는 주문에서 **리본 인쇄**로 따로 뽑습니다.
- **모바일·안드로이드에서는 리본 인쇄 불가** (PC 전용).
- 주문에 넣은 리본 문구는 **리본 인쇄** 화면에 **자동 채워짐** (수동 입력 불필요).
- 상세: **PR 단계** `#nav-printer`, `#nav-ribbon-from-order`

## 2. 주문 저장 → 그 밖에 자동으로 이어지는 것

| 트리거 | 이어지는 곳 | 사장님 눈에 보이는 결과 |
|--------|-------------|-------------------------|
| 주문 저장 + 고객 선택 | **고객 CRM** | 구매 횟수·총 구매액·마지막 주문일·포인트 갱신 |
| 주문 저장 + 재고 연동 상품 | **재고** | 수량 차감 (설정·상품에 따라) |
| 배송 화면에서 **실제 배송비** 저장 | **지출** | 배송비 지출 항목 자동 반영 |
| 배송 **완료** 처리 | **주문 상태** · 배송 목록 | 처리중 → 완료 |
| **일일 마감 정산** 실행 | 당일 **결제·미수·배송비 지출** 집계 | 금고 예상 잔액과 연동 |
| **외부발주** 저장 | 주문 `outsource_info` | 외주처·금액·인수증 문구 기록 |
| **지점 이관** 요청 | `order_transfers` | 받는 지점 주문 목록에 표시 |
| **회원사 수발주** 발주 | `external_orders` | 상대 회원사 수신함 |

매뉴얼 앵커: `#integration-order-crm` · `#integration-delivery-expense` · `#nav-order-flows`

## 3. 환경 설정이 먹히는 연동

| 설정 탭 | 주문·운영에 미치는 영향 |
|---------|-------------------------|
| **상점 정보** | 영수증·주문서 **매장명·전화·로고** |
| **주문/할인/포인트** | 새 주문 기본 할인·포인트 적립/사용 규칙 |
| **배송** | 새 주문 **배송비 자동 계산** (지역별·무료 기준) |
| **카테고리** | 상품·자재·지출 분류 전 화면 |
| **프린터/브릿지** | 저장 시 **자동 인쇄** on/off · 프린터 선택 |
| **회원사 수발주** | 회원사 발주 메뉴·네트워크 on/off |
