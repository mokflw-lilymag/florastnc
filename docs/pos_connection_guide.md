# 📟 FloraSync 포스(POS) 연동 가이드

안녕하세요! FloraSync와 POS(이지체크, 토스 등)를 연동하여 매장 운영을 자동화하는 가장 쉬운 방법을 안내해 드립니다. 본 가이드를 따라 하시면 **단 5분 만에** 실시간 주문 및 포인트 연동을 완료할 수 있습니다.

---

## 🚀 연동 전 준비사항
시작하기 전에 다음 항목이 준비되었는지 확인해 주세요.
*   **POS 단말기**: 현재 사용 중인 포스기 (이지체크 또는 토스 POS)
*   **관리자 계정**: POS 업체(VAN사) 관리자 포털 로그인 정보
*   **FloraSync 계정**: 매장의 FloraSync 대시보드 접근 권한

---

## 🛠️ 단계별 연동 순서

### 1단계: FloraSync POS 설정 메뉴 접속
1.  **FloraSync 대시보드**에 로그인합니다.
2.  왼쪽 메뉴에서 **'환경 설정' (⚙️ Settings)**을 클릭합니다.
3.  상단 탭 중 **'연동 및 자동화'**를 선택한 후, **'POS 연동'** 카드를 클릭하여 상세 페이지로 이동합니다.

### 2단계: 연동할 POS 종류 선택 및 정보 입력
1.  본인의 매장에서 사용 중인 POS 종류를 선택합니다. (예: `이지체크`, `토스 POS`)
2.  POS 업체로부터 발급받은 **상점 코드(Store Code)** 및 **API Key/Secret**을 입력합니다.
    > [!TIP]
    > **정보 확인 방법**: 이지체크는 대리점에 문의하시거나 '박스(VACS) 관리자 포털'에서 확인 가능하며, 토스 POS는 '토스페이먼츠' 관리자 화면에서 확인하실 수 있습니다.
3.  **'연동 활성화'** 스위치를 켭니다.

### 3단계: 웹훅(Webhook) URL 복사 및 등록 (가장 중요!)
POS에서 결제가 일어날 때 FloraSync로 신호를 보내주기 위한 주소를 등록해야 합니다.
1.  FloraSync POS 설정 페이지 하단에 생성된 **'웹훅 수신 URL'**을 복사합니다.
2.  **POS 업체 관리자 포털**의 Webhook 설정 메뉴로 이동합니다.
3.  복사한 URL을 붙여넣고, **시크릿 키(Webhook Secret)**를 FloraSync 설정 창에도 동일하게 입력해 줍니다.
    *   *이지체크*: 대리점을 통해 '결제 알림 웹훅' 등록을 요청하세요.
    *   *토스 POS*: 개발자 센터의 '웹훅 설정'에서 복사한 주소를 입력하세요.

### 4단계: 포인트 및 고객 자동화 설정
1.  **포인트 적립률**: 결제 금액의 몇 %를 적립할지 입력합니다. (예: 1%, 3% 등)
2.  **비회원 자동 등록**: 포스에서 전화번호를 입력했을 때, FloraSync에 등록되지 않은 고객이라면 자동으로 신규 가입 처리를 할지 결정합니다. (`ON` 권장)
3.  **'설정 저장'** 버튼을 클릭하여 완료합니다.

### 5단계: 연동 테스트 (마지막 단계)
1.  단말기에서 100원 또는 1,000원 결제를 진행해 봅니다.
2.  **FloraSync 주문 목록**에 즉시 `📟 POS` 배지가 달린 주문이 생성되는지 확인합니다.
3.  **고객 관리** 메뉴에서 포인트가 정상적으로 적립되었는지 확인합니다.

---

## 🆘 도움이 필요하신가요?
연동 과정에서 어려움이 있다면 다음 경로로 문의해 주세요.
*   **FloraSync 기술 지원**: 010-1234-5678 (카톡/전화)
*   **POS 대리점**: 카드 결제 알림(Webhook) 전송 가능 여부를 확인해 주세요.

---

> [!IMPORTANT]  
> **주의사항**: 웹훅 URL은 외부로 노출되지 않도록 주의해 주세요. 주소가 노출되었을 경우 '재생성' 버튼을 눌러 주소를 새로 발급받을 수 있습니다.
