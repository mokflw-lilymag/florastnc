본 문서는 모든 버튼과 메뉴의 연동성을 심층적으로 다루는 공식 백과사전입니다.

1

환경 설정 (기초 공사)

이 단계가 플로라싱크의 핵심 중의 핵심입니다. 여기서 템플릿과 상품을 제대로 등록하지 않으면, AI 주문 입력이나 클라우드 원격 출력이 정상적으로 작동하지 않습니다.

1-1. 기본 정보 및 매장 설정

[ 환경설정 > 기본 정보 화면 캡처 ]

매장의 상호명, 대표 전화번호, 사업자등록번호, 그리고 영수증에 출력될 매장 로고를 업로드합니다.

  • 로고 이미지: 영수증/인수증 출력 시 상단에 흑백으로 인쇄됩니다. 고대비 이미지를 권장합니다.
  • 카카오톡 연동 키: 알림톡 발송을 위한 비즈메시지 키를 입력하는 곳입니다.

1-2. 카테고리/상품 마스터 (엑셀 연동)

[ 환경설정 > 상품 관리 화면 캡처 ]

※ AI 매칭의 기준점

이곳에 등록된 단어들(예: "축하화환 3단", "대형 바구니")을 기준으로 AI가 카카오톡 텍스트를 분석하여 상품을 자동 선택합니다. 최대한 자세히 적어주세요.

  • 개별 등록: [상품 추가] 버튼을 눌러 이름, 가격, 카테고리를 설정합니다.
  • 엑셀 대량 업로드: 우측 상단의 [양식 다운로드] 후 엑셀에서 수백 개의 상품을 작성하고 [엑셀 업로드]를 누르면 1초 만에 DB가 구축됩니다.

1-3. 디자인 템플릿 에디터

[ 템플릿 에디터 드래그 앤 드롭 화면 캡처 ]

리본(우/좌), 메시지 카드의 디자인을 세팅합니다. 드래그 앤 드롭(Drag & Drop)을 완벽 지원합니다.

  • 폰트 및 크기: 텍스트 상자를 클릭하고 우측 속성창에서 폰트(나눔고딕 등)와 폰트 크기를 설정합니다.
  • 변수 매핑: {receiver_message}, {sender_message} 같은 변수 블록을 원하는 위치에 배치하세요. 실제 출력 시 이 위치에 주문서의 문구가 자동으로 치환되어 출력됩니다.

1-4. 프린터 세팅 (수동 등록 필요 없음!)

💡 웹사이트에 프린터를 일일이 타자로 등록해야 하나요?

아닙니다! IP 주소나 프린터 이름을 직접 타이핑해서 등록하실 필요가 전혀 없습니다.

프린터가 꽂혀있는 매장 PC에 '리본니스트 브릿지(Ribbonist Bridge)'라는 전용 프로그램만 켜두세요. 이 브릿지 프로그램이 매장 컴퓨터에 깔린 모든 프린터 목록을 스스로 싹 읽어내서 플로라싱크 클라우드로 알아서 올려보내 줍니다.

사장님은 그저 플로라싱크 화면에 뜬 드롭다운 목록을 누르고 "리본은 A프린터로, 영수증은 B프린터로 뽑아라" 라고 마우스로 클릭해서 지정(매핑)만 해주시면 끝납니다.

영수증/인수증 프린터 지정

브릿지가 자동으로 찾아준 목록에서 80mm/58mm 감열식 영수증 프린터를 고르기만 하세요. 마진과 절삭 옵션도 여기서 간단히 클릭으로 제어합니다.

리본 전용 프린터 지정

마찬가지로 목록에서 리본 전용 프린터를 선택하고, 용지 너비(100mm, 150mm 등)를 지정해주면 모든 세팅이 완료됩니다.

1-5. 쇼핑몰 연동 실전 가이드 (네이버/카페24)

스마트스토어와 카페24의 주문을 플로라싱크로 실시간으로 가져오는 마법의 첫걸음입니다. 각 쇼핑몰의 연동 방법을 순서대로 천천히 따라와 주세요.

A. 네이버 커머스 (스마트스토어) 연동하기

1 네이버 커머스 API 센터에서 '애플리케이션' 생성하기
  1. 네이버 커머스 API 센터에 접속하여 스마트스토어 판매자 아이디로 로그인합니다.
  2. 좌측 메뉴에서 [애플리케이션 관리] -> [애플리케이션 등록] 버튼을 누릅니다.
  3. 앱 이름은 '플로라싱크'로 적어주시고, 아래의 API 권한 선택에서 [주문], [상품] 관련 권한을 필수로 체크해 줍니다.
  4. 등록을 완료하면 Application ID와 Application Secret 코드가 발급됩니다. 이 창을 닫지 말고 열어두세요.
2 플로라싱크 앱에 정보 입력하고 저장하기
  1. 플로라싱크 [환경 설정] -> [쇼핑몰 연동 관리] 메뉴에서 '네이버 커머스 API' 스위치를 켭니다.
  2. 발급받은 Application ID와 Application Secret을 빈칸에 정확히 복사해서 붙여넣습니다.
  3. 우측 하단의 [인증 테스트 및 저장] 버튼을 누릅니다. 네이버는 재인증 과정 없이 이것으로 모든 연동이 끝납니다!

B. 카페24 자사몰 연동하기

1 카페24 개발자센터에서 '열쇠(API 키)' 발급받기

가장 먼저, 카페24 쇼핑몰의 주문 정보를 플로라싱크가 읽어올 수 있도록 허락(권한)을 받아야 합니다.

  1. 카페24 개발자센터에 접속하여 쇼핑몰 관리자 아이디로 로그인합니다.
  2. 상단 메뉴에서 [나의 앱] 클릭 후, 우측 상단의 [앱 만들기] 버튼을 누릅니다. (앱 이름은 '플로라싱크 연동'으로 자유롭게 입력하세요)
  3. 가장 중요한 단계! 앱 설정 화면에서 [API 권한] 탭을 클릭하고, 수많은 권한 목록 중 mall.read_order (주문 읽기)를 찾아 반드시 체크합니다.
  4. 설정을 저장하면 Client ID와 Client Secret이라는 두 개의 긴 영문/숫자 코드가 발급됩니다. 이 창을 그대로 열어두세요.
2 플로라싱크 앱에 열쇠(키) 입력하고 저장하기

이제 플로라싱크로 돌아와서 아까 받은 열쇠를 꽂아줄 차례입니다.

[ 환경설정 > 카페24 연동 관리 화면 캡처 자리 ]
  1. 플로라싱크 [환경 설정] -> [쇼핑몰 연동 관리] 메뉴에서 '카페24 개발자 API' 스위치를 켭니다.
  2. 아래 3개의 빈칸을 정확히 채워 넣습니다:
    • 쇼핑몰 아이디 (Mall ID): 사장님의 카페24 관리자 아이디 (예: florasync)
    • Client ID: 1단계에서 띄워둔 Client ID를 복사 & 붙여넣기
    • Client Secret: 1단계에서 띄워둔 Client Secret을 복사 & 붙여넣기
  3. 작성이 끝났다면 우측 하단의 [인증 테스트 및 저장] 버튼을 누릅니다. (설정이 저장되었습니다 메시지 확인)
3 마지막 관문: [재인증(로그인 연동)] 버튼 누르기

설정이 저장되었다면 마지막으로 카페24 본진의 승인을 받아야 합니다.

  1. 키 입력칸 바로 위에 있는 초록색 글씨의 [카페24 로그인 연동] (또는 [재인증]) 버튼을 클릭합니다.
  2. 팝업창으로 카페24 로그인 화면이 뜨면, 쇼핑몰 관리자 아이디/비밀번호를 입력하고 로그인합니다.
  3. 권한 허용 창이 나오면 [동의함]을 눌러줍니다.
  4. 팝업창이 닫히며 "성공적으로 연동되었습니다" 메시지가 뜨면 완벽하게 끝났습니다! 🎉
  5. 이제 좌측 [주문 관리] 메뉴로 가셔서 우측 상단의 [쇼핑몰 동기화] 버튼을 누르시면, 방금 연동한 카페24의 주문들이 촤르륵 쏟아져 들어옵니다.

3 독보적 킬러 기능: 범용 프린터 완벽 호환 (추가 비용 ZERO) 🖨️

"비싼 리본 전용 프린터? 이제 더 이상 사지 마세요."

리본 하나 뽑자고 수백만 원짜리 전용 장비를 구매하실 필요가 없습니다. 플로라싱크의 강력한 '프린트 브릿지(Print Bridge)' 기술은 매장에서 이미 사용 중인 평범한 프린터를 완벽한 리본 프린터로 변신시킵니다.

🖨️
Epson M시리즈

꽃집의 필수품 M105 등. 흑백 전용 잉크젯 프린터로 선명한 리본 출력을 완벽하게 지원합니다.

🎨
Epson L시리즈

L210 등 컬러 라인업 호환 완료. 원하는 색상으로 리본을 예쁘게 디자인해 출력할 수 있습니다.

🧾
감열식 영수증 (Xprint)

XP-DT108B 등 저렴한 감열식 프린터를 이용해 송장 및 인수증을 빠르고 끊김 없이 출력합니다.

💡 플로라싱크만의 장비 투자비 제로(0) 매직

저희는 꽃집 사장님들이 가장 많이 사용하시는 프린터 기종을 수없이 테스트하며 브릿지 기술을 최적화했습니다. 이제 값비싼 전용 기계와 전용 소모품 유지비에서 완전히 해방되세요. 플로라싱크를 이용하면 그저 "저렴한 리본 공용지만 사서, 원래 쓰던 프린터에 넣고 바로 출력"하면 끝입니다. 불필요한 추가 장비 투자 없이, 이미 매장에 있는 기계를 200% 활용하세요.

🔥 플로라싱크의 14일 토큰 무한 갱신 마법 (카페24)

보안상 카페24의 열쇠(접근 토큰)는 원래 2시간마다 만료됩니다. 플로라싱크가 아닌 다른 프로그램을 쓰신다면, 2시간마다 매번 로그인(재인증)을 새로 해주셔야만 주문이 수집됩니다. 하지만 플로라싱크는 다릅니다.

플로라싱크의 서버는 2시간 만료 시점에 맞춰 'Refresh Token(14일짜리 스페어키)'을 사용하여 사장님 몰래 새 열쇠로 0.1초 만에 자동 갱신합니다. 동시에 이 14일짜리 스페어키의 기한도 계속 새것으로 연장받기 때문에, 사장님께서 해외여행을 가시느라 플로라싱크 앱을 14일(336시간) 넘게 아예 접속조차 안 하는 극단적인 상황이 아니라면, 앞으로 두 번 다시 [재인증] 버튼을 누르실 필요가 없습니다!

2

주문 관리 및 AI 기능

온라인 주문은 자동으로 들어오고, 카톡이나 문자로 온 주문은 AI가 눈 깜짝할 사이에 처리합니다.

🤖 2-1. AI 원클릭 주문 입력

[ AI 주문 모달창 캡처 - 텍스트를 붙여넣은 상태 ]

우측 상단의 보라색 [AI 주문 입력] 버튼을 누릅니다. 커다란 텍스트 박스에 고객이 보낸 비정형 카톡 메시지를 그대로 복사해서 붙여넣고 [AI 분석] 버튼을 누릅니다.

입력된 카톡 텍스트

"내일 오전 10시까지 강남구 역삼동 스타빌딩 3층 김사장님께 10만원 축하 화환 부탁해요. 문구는 축 개업 / 친구 홍길동"

AI 분석 결과 (자동 매핑)
  • • 일시: 2026-XX-XX 10:00
  • • 주소: 강남구 역삼동 스타빌딩 3층
  • • 상품: 축하화환 (환경설정 기준 매칭)
  • • 우측 리본: 축 개업
  • • 좌측 리본: 친구 홍길동

🔍 2-3. 주문 상세 모달창 (관제탑)

[ 주문 상세 모달 전체 화면 캡처 ]

주문 리스트에서 한 줄을 클릭하면 이 모달창이 열립니다. 여기서 모든 작업이 이루어집니다.

  • 주문 정보 수정: 금액(배송비/할인금액), 주소, 메모 등을 즉석에서 수정하고 [저장]합니다. (금액은 0원 방지 로직이 완벽하게 적용되어 있습니다.)
  • 상태 변경: 우측 상단의 배지를 눌러 [미결] -> [처리중] -> [배송완료]로 프로세스를 넘깁니다.
  • 사진 등록 및 알림톡: 제작이 완료된 꽃 사진을 드래그 앤 드롭으로 업로드한 뒤, 노란색 [카카오톡 알림 발송] 버튼을 누르면 고객에게 완성 알림톡이 전송됩니다.

CRM (고객 관리) 및 캘린더

4-1. 강력한 단골 관리 (CRM)

[ 고객 관리 리스트 캡처 ]

주문이 들어오면 고객 DB가 자동으로 쌓입니다. 특정 고객을 클릭하면 과거에 언제, 어떤 꽃을 얼마에 주문했는지, 메모는 무엇이었는지 타임라인 형태로 모두 볼 수 있어 단골 관리에 탁월합니다.

4-2. 직관적인 캘린더 뷰

[ 한 달 캘린더 뷰 캡처 ]

월별/주별 캘린더를 통해 특정 날짜(예: 어버이날, 졸업식 시즌)에 몰려 있는 예약 주문건을 색상 배지로 한눈에 파악합니다. 배송 일정이 변경되면 마우스로 드래그 앤 드롭하여 날짜를 즉시 이동시킬 수 있습니다.

미래 비전: 오프라인 POS 실시간 연동

[ POS 기기와 웹 화면이 연결되는 3D 다이어그램 ]

온·오프라인 매출 데이터의 거대한 통합

사장님께서 현재 웹에서 열심히 등록하고 계신 카테고리와 상품 데이터베이스, 그리고 쌓여가는 고객 데이터는 플로라싱크 생태계의 귀중한 자산입니다.

머지않아 출시될 플로라싱크 전용 스마트 POS 기기는 이 웹 데이터와 0.1초 단위로 실시간 동기화됩니다. 워크인(매장 방문) 고객의 카드 결제와 온라인 쇼핑몰 주문이 하나의 대시보드에서 완벽하게 합쳐지는 꽃집 비즈니스의 최종 진화 형태를 경험하게 될 것입니다.
