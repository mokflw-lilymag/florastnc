## 시스템 개요

릴리매거 ERP는 **꽃집 프랜차이즈**에서 주문·고객·재고·비용을 한곳에서 다루기 위한 **웹 기반** 업무 시스템입니다. 인터넷이 되는 PC나 태블릿의 **브라우저**(Chrome, Edge 등)로 접속합니다.

### 주요 기능 (한눈에)

| 구분 | 할 수 있는 일 |
|------|----------------|
| **주문** | 신규 접수, 목록·상태 확인, 픽업/배송 일정 |
| **고객·수령** | 고객 등록·검색, 수령자 정보 관리 |
| **자재·재고** | 본사에 자재 요청, 매장 자재·재고 확인, 바코드 조회 |
| **비용·분석** | 간편 지출, 대시보드·통계(역할에 따라 다름) |
| **기타** | 샘플 앨범, 거래처, 인사·설정 등(메뉴는 권한별) |

### 사용자 역할 (메뉴가 달라집니다)

- **본사 관리자**: 모든 지점 데이터 조회·설정, 구매·상품·지점 관리 등 **전체 기능**에 접근할 수 있습니다.
- **지점 관리자**: **우리 지점** 중심으로 주문·고객·자재 등 대부분의 매장 업무를 처리합니다. 본사 전용 메뉴는 제한될 수 있습니다.
- **지점 직원**: 역할에 따라 **주문·자재·고객** 등 지정된 메뉴만 보일 수 있습니다.

> 로그인 후 **왼쪽 메뉴**에 실제로 보이는 항목이 곧 사용 가능한 기능입니다. 메뉴에 없으면 권한이 없는 것이니, 본사 관리자에게 요청하세요.

## 로그인 및 기본 사용법

### 로그인하기

1. **브라우저 주소창**에 회사에서 안내한 **시스템 주소(URL)** 를 입력하고 접속합니다.  
2. **이메일**과 **비밀번호**를 입력합니다. (보통 회사에서 받은 계정입니다.)  
3. **로그인** 버튼을 클릭합니다.  
4. 잠시 후 **대시보드** 또는 권한에 맞는 첫 화면으로 들어갑니다.

**잘 안 될 때**

- 비밀번호는 **대·소문자를 구분**합니다. Caps Lock이 켜져 있지 않은지 확인하세요.  
- 이메일에 **오타**(공백, `@` 앞뒤)가 없는지 다시 확인하세요.  
- 여러 번 틀리면 보안을 위해 잠길 수 있습니다. 그때는 **본사·관리자**에게 계정 상태를 문의하세요.  
- **다른 사람 PC**에서는 사용 후 반드시 **로그아웃**하세요.

### 화면 구성 (공통)

| 영역 | 역할 |
|------|------|
| **왼쪽 사이드바** | 메뉴 목록. 여기서 이동할 페이지를 고릅니다. |
| **위쪽 헤더** | 현재 페이지 제목, 알림, (있을 경우) 지점·검색 등 |
| **가운데 본문** | 목록, 입력 폼, 버튼 등 실제 업무 내용 |

많은 화면에는 **날짜**, **지점**, **검색어**, **상태** 같은 **필터**가 있습니다. 데이터가 안 보이면 먼저 필터가 너무 좁게 잡혀 있는지 확인해 보세요.

### 로그아웃

1. 왼쪽 사이드바 **아래쪽**으로 스크롤합니다.  
2. **로그아웃**을 클릭합니다.  
3. 로그인 화면으로 돌아가면 완료입니다. **공용 PC**에서는 반드시 로그아웃하세요.

### 용어 간단 정리

- **지점 / 매장**: 소속된 가게 단위. 지점 사용자는 보통 **자기 지점** 데이터만 봅니다.  
- **주문**: 고객에게 판매하는 한 건의 접수·처리 단위.  
- **자재**: 꽃·포장재 등 매장에서 쓰는 물품. 본사에 **요청**하면 배송 등으로 받습니다.

## 지점 사용자 메뉴얼

지점 사용자는 다음과 같은 메뉴에 접근할 수 있습니다:

### 1. 샘플앨범
**스크린샷 위치**: 샘플앨범 메인 페이지 (`/dashboard/sample-albums`)

**기능 설명**:
샘플앨범 메뉴는 꽃다발, 꽃바구니 등 다양한 제품의 샘플 사진을 카테고리별로 관리하고, 고객에게 제품 미리보기를 제공하며, 새로운 샘플을 추가하거나 기존 샘플을 수정할 수 있는 기능을 제공합니다.

**사용 방법**:

1.  **샘플앨범 페이지 접속**:
    사이드바 메뉴에서 `샘플앨범`을 클릭하여 샘플앨범 메인 페이지로 이동합니다.
    ![샘플앨범 메인 페이지](placeholder_sample_albums_main.png)

2.  **카테고리별 필터링**:
    페이지 상단에 있는 카테고리 드롭다운 메뉴를 사용하여 원하는 제품 유형(예: 꽃다발, 꽃바구니, 화분 등)을 선택하여 샘플을 필터링할 수 있습니다.
    ![카테고리 필터링](placeholder_sample_albums_filter.png)

3.  **샘플 상세 보기**:
    목록에서 특정 샘플 이미지를 클릭하면 해당 샘플의 상세 정보(이름, 설명, 추가 이미지 등)를 확인할 수 있습니다.
    ![샘플 상세 보기](placeholder_sample_albums_detail.png)

4.  **새 앨범 만들기**:
    새로운 샘플 앨범을 추가하려면 페이지 우측 상단에 있는 `"새 앨범 만들기"` 버튼을 클릭합니다.
    ![새 앨범 만들기 버튼](placeholder_sample_albums_new_button.png)
    새 앨범 생성 팝업이 나타나면 앨범 이름, 카테고리, 설명 등을 입력하고 이미지를 업로드한 후 `"저장"` 버튼을 클릭합니다.
    ![새 앨범 만들기 팝업](placeholder_sample_albums_new_popup.png)

5.  **기존 샘플 수정/삭제**:
    기존 샘플을 수정하거나 삭제하려면 샘플 상세 보기 페이지에서 해당 옵션을 선택합니다.
    ![샘플 수정/삭제](placeholder_sample_albums_edit_delete.png)

### 2. 주문 접수
**스크린샷 위치**: 주문 접수 페이지 (`/dashboard/orders/new`)

**이 메뉴로 하는 일**  
고객 주문을 **처음부터 끝까지** 시스템에 넣습니다. 저장이 끝나면 **주문 현황** 목록에 나타납니다.

**준비하면 좋은 것**  
고객 연락처, 원하시는 상품(또는 비슷한 샘플), 픽업/배송 일시, 특별 요청(카드 문구, 색상 등).

**사용 방법 (권장 순서)**  

1. **메뉴 이동**  
   왼쪽 사이드바에서 **주문 접수**·**새 주문** 등 주문을 새로 만드는 메뉴를 클릭합니다.

2. **고객**  
   - 이미 등록된 고객이면 **이름·전화번호** 등으로 검색해 선택합니다.  
   - 신규 고객이면 **새 고객**으로 필요한 정보를 입력합니다. (필수 항목은 화면에 표시된 대로)

3. **상품·금액**  
   상품을 고르고 **수량·옵션**을 맞춥니다. 여러 품목이면 행을 추가합니다.

4. **전달 방법**  
   **픽업**(매장 수령)인지 **배송**인지 선택하고, 날짜·시간대·주소(배송 시)를 넣습니다.

5. **메모**  
   포장·리본·카드 문구 등 **특이사항**은 메모란에 적어 두면 이후 작업자가 확인하기 좋습니다.

6. **저장**  
   내용을 다시 확인한 뒤 **저장**·**주문 등록** 버튼을 누릅니다.  
   저장 후 **주문 번호**나 목록에서 해당 건이 보이는지 확인하면 안심할 수 있습니다.

> **팁**: 중간에 나가면 입력이 사라질 수 있으니, 긴 주문은 메모장에 먼저 적어 두고 옮겨 적는 것도 방법입니다.

### 3. 주문 현황
**스크린샷 위치**: 주문 현황 페이지 (`/dashboard/orders`)

**이 메뉴로 하는 일**  
이미 넣은 주문을 **목록으로 보고**, **상세 내용을 확인**하며, 권한이 있으면 **상태 변경·수정**을 합니다.

**화면에서 보통 보는 것**  
날짜, 주문 상태, 고객명, 금액, 픽업/배송 정보 등이 한 줄에 요약되어 있습니다.

**사용 방법**  

1. **필터로 범위 줄이기**  
   - **날짜**: 오늘·이번 주 등 업무에 맞게 설정합니다.  
   - **상태**: 진행중·완료·취소 등으로 좁힐 수 있으면 활용합니다.  
   - **검색**: 고객명·전화번호 일부로 찾습니다.

2. **한 건 열어보기**  
   목록에서 주문 **한 줄을 클릭**하거나 **상세**·**보기** 버튼을 눌러 상세 화면으로 갑니다.

3. **상태·내용 변경**  
   권한과 정책에 따라 **상태 변경**, **수정**, **취소** 등이 가능합니다. 버튼 이름은 버전에 따라 다를 수 있습니다.

4. **출력·메시지**  
   매장 정책에 따라 **인쇄**, **메시지 발송** 등이 상세 화면 또는 메뉴에 있을 수 있습니다.

> **데이터가 안 보일 때**: 날짜 필터가 너무 좁지 않은지, 지점 필터가 우리 매장인지 먼저 확인하세요. ([빠른 시작 가이드](quick-start-guide.md) 참고)

## 본사 관리자 메뉴얼

본사 관리자는 지점 사용자의 모든 기능에 더해 다음과 같은 추가 메뉴에 접근할 수 있습니다:

### 1. 대시보드
**스크린샷 위치**: 대시보드 메인 페이지 (`/dashboard`)

**기능 설명**:
- 전체 시스템 현황 요약
- 주요 지표 및 통계 표시
- 최근 활동 내역

**사용 방법**:
1. 전체 매출 및 주문 현황 확인
2. 지점별 성과 지표 확인
3. 최근 등록된 주문 및 요청 확인

### 2. 상품 관리
**스크린샷 위치**: 상품 관리 페이지 (`/dashboard/products`)

**기능 설명**:
- 판매 상품 등록 및 관리
- 상품 카테고리 설정
- 가격 및 재고 정보 관리

**사용 방법**:
1. 새 상품 등록
2. 기존 상품 정보 수정
3. 상품별 재고 및 가격 설정

### 3. 재고 변동 기록
**스크린샷 위치**: 재고 변동 기록 페이지 (`/dashboard/stock-history`)

**기능 설명**:
- 모든 지점의 재고 변동 이력 조회
- 입출고 내역 추적
- 재고 분석 및 리포트

**사용 방법**:
1. 기간별 재고 변동 이력 조회
2. 지점별 재고 현황 확인
3. 재고 분석 리포트 생성

### 4. 구매 관리
**스크린샷 위치**: 구매 관리 페이지 (`/dashboard/purchase-management`)

**기능 설명**:
- 지점별 자재 요청 통합 관리
- 구매 계획 수립
- 공급업체 관리

**사용 방법**:
1. 지점별 자재 요청 확인
2. 구매 계획 수립 및 실행
3. 공급업체별 주문 관리

### 5. 지점 관리
**스크린샷 위치**: 지점 관리 페이지 (`/dashboard/branches`)

**기능 설명**:
- 가맹점 지점 정보 관리
- 지점별 설정 및 권한 관리
- 지점 성과 모니터링

**사용 방법**:
1. 새 지점 등록
2. 지점 정보 수정
3. 지점별 성과 지표 확인

### 6. 비용 관리
**스크린샷 위치**: 비용 관리 페이지 (`/dashboard/expenses`)

**기능 설명**:
- 전체 비용 내역 관리
- 비용 승인 프로세스
- 비용 분석 및 리포트

**사용 방법**:
1. 비용 요청 승인/반려
2. 비용 내역 조회 및 분석
3. 비용 리포트 생성

### 7. 예산 관리
**스크린샷 위치**: 예산 관리 페이지 (`/dashboard/budgets`)

**기능 설명**:
- 지점별 예산 설정 및 관리
- 예산 대비 실적 분석
- 예산 알림 설정

**사용 방법**:
1. 지점별 예산 설정
2. 예산 대비 실적 확인
3. 예산 초과 알림 설정

### 8. 리포트 분석
**스크린샷 위치**: 리포트 분석 페이지 (`/dashboard/reports`)

**기능 설명**:
- 종합적인 비즈니스 분석
- 매출, 주문, 재고 등 다양한 리포트
- 데이터 시각화

**사용 방법**:
1. 원하는 리포트 유형 선택
2. 기간 및 필터 설정
3. 리포트 생성 및 다운로드

### 9. 인사 관리
**스크린샷 위치**: 인사 관리 페이지 (`/dashboard/hr`)

**기능 설명**:
- 직원 정보 관리
- 근무 기록 관리
- 인사 관련 문서 관리

**사용 방법**:
1. 직원 정보 등록 및 수정
2. 근무 기록 관리
3. 인사 관련 통계 확인

### 10. 사용자 관리
**스크린샷 위치**: 사용자 관리 페이지 (`/dashboard/users`)

**기능 설명**:
- 시스템 사용자 계정 관리
- 사용자 권한 설정
- 로그인 이력 관리

**사용 방법**:
1. 새 사용자 계정 생성
2. 사용자 권한 수정
3. 사용자 활동 이력 확인

### 11. 시스템 설정
**스크린샷 위치**: 시스템 설정 페이지 (`/dashboard/settings`)

**기능 설명**:
- 시스템 전반적인 설정 관리
- 백업 및 복원
- 시스템 환경 설정

**사용 방법**:
1. 시스템 기본 설정 변경
2. 데이터 백업 및 복원
3. 시스템 환경 최적화

## 공통 기능

### 검색 및 필터링
대부분의 목록 페이지에서 검색 및 필터링 기능을 제공합니다:
- 텍스트 검색
- 날짜 범위 필터
- 상태별 필터
- 카테고리별 필터

### 데이터 내보내기
많은 페이지에서 Excel 형식으로 데이터를 내보낼 수 있습니다:
1. "내보내기" 버튼 클릭
2. 내보낼 데이터 범위 선택
3. Excel 파일 다운로드

### 인쇄 기능
주문서, 라벨 등은 인쇄 기능을 제공합니다:
1. "인쇄" 버튼 클릭
2. 인쇄 미리보기 확인
3. 인쇄 실행

### 알림 및 메시지
시스템은 다음과 같은 알림을 제공합니다:
- 작업 완료 메시지
- 오류 알림
- 상태 변경 알림
