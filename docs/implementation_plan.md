# Floxync SaaS 차별화 전략 마스터 플랜

> **최종 수정일:** 2026-04-05
> **핵심 원칙:** 본사 = 소프트웨어만 제공. 돈의 흐름에 일절 관여하지 않음.

---

## 1. 개요

Floxync를 **"화원 사장님의 두 번째 뇌"**로 격상시키기 위한 차별화 전략. 전 세계 화원 소프트웨어(Hana POS, FloristWare, Curate 등) 분석 결과를 반영한 종합 로드맵.

---

| # | 수익원 | 설명 | 시기 |
|---|--------|------|------|
| 1 | **SaaS 월 구독료** | Free / ERP / Ribbon / Pro 플랜 월정액 | 현재 |

## 3. 본사 운영 업무

> [!IMPORTANT]
> 본사 = 소프트웨어 + AI만 제공. 돈의 흐름에 일절 관여하지 않음.

### 본사가 관리하는 항목

| # | 항목 | 내용 | 주기 |
|---|------|------|:---:|
| 1 | **AI API 키** (Gemini) | 본사 1개 키로 전 매장 AI 기능 제공 | 상시 |
| 2 | **식물 DB** | 물주기 카드용 200~300종 데이터 | 월 1회 추가 |
| 3 | **디자인 템플릿** | 메시지카드 배경, 물주기카드 디자인 5종 | 시즌별 |
| 4 | **매장 API 연결 지원** | 아래 3단계 지원 체계 운영 | 상시 |
| 5 | **소프트웨어 업데이트** | 기능 추가, 버그 수정, 보안 패치 | 상시 |

### 매장 API 연결 지원: 3단계 체계

**1단계: 셀프 서비스 (80% 매장 — 전화 안 옴)**
- 앱 내 각 연동별 단계별 가이드 (스크린샷 포함)
- 3분 영상 튜토리얼 (유튜브)
- [연결 테스트] 버튼으로 즉시 확인
```
환경설정 > 토스페이먼츠 > [📖 연결 가이드 보기]
→ Step 1~4 스크린샷 안내 → 키 붙여넣기 → [테스트] → ✅ 완료
```

**2단계: 원격 설정 지원 (15% 매장 — 키만 받으면 2~3분)**
- 매장 사장님이 API 키만 발급 (사업자 인증은 본인만 가능)
- 키를 본사에 전달 → 슈퍼 어드민에서 대신 입력 + 테스트
```
본사 관리자 포털 > 매장 선택 > [원격 설정 지원] > 키 입력 > [테스트] > 완료
```

**3단계: 방문/전화 상세 지원 (5% 매장 — 고령 사장님 등)**
- 전화 통화하며 화면 공유로 안내
- 필요 시 유료 방문 설정 서비스

## 4. 외부 서비스 연동

> [!IMPORTANT]
> 모든 API = 각 매장이 자기 API 키 입력. 본사는 코드만 제공.

### 연동 요약표

| 연동 대상 | 발급 주체 | 매장 할 일 | 비용 부담 |
|-----------|:---:|:---:|:---:|

| 카페24 | 각 매장 | Key 입력 | 각 매장 |
| AI 기능 전체 | **본사 1개** | **ON/OFF만** | **구독료 포함** |

## 13. 인쇄 시스템 (Global Gateway)

### 13-A. 리본 프린트
- 유인: 다양한 폭 / 무인: 1가지 고정

### 13-B. 폼택 인쇄 - 자동 칸 이동
- 뒤에서 앞으로 자동 이동, 직원 수동 지정 불필요

### 13-C. 프리미엄 메시지 카드 (신규)
- 배경 템플릿 다양 + 자동 줄바꿈 + 폰트 자동 조절 + 양면 인쇄

### 13-D. 식물 관리카드 (물주기 카드) 인쇄
- 본사에서 식물 DB + 친근한 문구 기본 제공 → 매장이 재편집 가능 → A4에 12~20장 출력

### 13-E. Anywhere Cloud Printing (언제 어디서나 인쇄)
> [!TIP]
> "드라이버 설정의 번거로움을 제거하고, 전 세계 어디서든 버튼 하나로 인쇄하는 마법"

- **기술적 정의**: 클라우드 기반 인쇄 릴레이(Cloud Printing Relay) 시스템.
- **작동 방식**:
  1. **요청**: 모바일 브라우저/앱에서 '인쇄' 클릭 → 데이터가 Supabase 인쇄 대기열(Queue)에 적재.
  2. **감지**: 매장 PC에서 실행 중인 `Printer Bridge`가 1~2초 간격으로 대기열 폴링/실시간 감지.
  3. **출력**: Bridge가 데이터를 내려받아 로컬 프린터로 즉시 전송 및 출력.
- **핵심 가치**:
  - **공간 제약 해제**: 사장님이 새벽 시장에서 물건을 사며 스마트폰으로 인쇄를 누르면, 매장 직원은 리본이 나오는 것을 보고 미리 준비 가능.
  - **설정 제약 해제**: 복잡한 드라이버 설치, 포워딩, 고정 IP 설정 등이 필요 없음. 오직 '사장님 계정 로그인'만으로 프린터와 연결.
  - **멀티 디바이스**: 태블릿, 안드로이드, 아이폰 등 어떤 브라우저에서도 즉시 인쇄 가능.

### 13-F. 글로벌 하드웨워 무관(Hardware Agnostic) 전략
> [!IMPORTANT]
> Floxync는 장비를 파는 회사가 아닙니다. 이미 매장에 있는 장비를 똑똑하게 연결하는 플랫폼입니다.

- **범용 하드웨어 지원**:
  - **XPrint (106mm~170mm) 지원**: 고가의 전용 리본 프린터(정원인더스트리 등) 대신, 10만 원대 범용 감열 프린터(XPrint 등)로도 고품질 인쇄가 가능하게 최적화.
  - **Epson & HP**: 기존 매장에서 보유한 일반 잉크젯/레이저 프린터 완벽 지원.
- **경제성 분석**:
  - **본체 비용**: 기존 39만 원대 → 11만 원대 (약 72% 절감).
  - **소소품**: 전용 카트리지 대신 범용 리본/감열지 사용으로 유지비 50% 이상 절감.
- **국가별 최적화 (Localization)**:
  - **한국/일본**: 리본 및 라벨 프린팅 중심.
  - **동남아 (베트남 등)**: 대형 화환용 패널(Panel) 인쇄 템플릿 자동 매핑.
  - **북미/유럽**: 프리미엄 메시지 카드 및 배너 프린팅 자동화.

## 18. 구현 우선순위 로드맵

### Phase 1: 기반 구축 (완료 및 진행 중)
- 로컬 Printer Bridge (v25.0) 안정화.
- 폼택 및 메시지 카드 템플릿 엔진 구축.
- 정원인더스트리 등 기존 하드웨어 연동 확인.

### Phase 2: 클라우드 릴레이 & 경제성 (3~6주)
- **Supabase 인쇄 대기열(Queue) 및 폴링 로직 구현.**
- **Anywhere Cloud Printing 모바일 UI/UX 적용.**
- **XPrint 등 범용 장비용 인쇄 드라이버 라이브러리 확장.**
- 식물 관리카드 및 폼택 자동 칸 이동 고도화.

### Phase 3: 글로벌 스케일링 (7~12주)
- **국가별 인쇄 템플릿 자동 매핑 시스템.**
- **글로벌 하드웨어 드라이버 클라우드 라이브러리 구축.**
- SNS 자동 마케팅 및 AI 맞춤 상품 추천 연동.
- 무인 키오스크용 클라우드 인쇄 모듈 통합.

## 19. 검증 계획

- **Cloud Relay 테스트**: 모바일(LTE) 환경에서 인쇄 요청 시 매장 PC에서 5초 이내 출력 확인.
- **Hardware Agnostic**: Epson, 정원, XPrint 등 3종 이상의 프린터에서 동일 리본 퀄리티 확인.
- **글로벌 템플릿**: 베트남어(화환 패널) 및 영어(카드) 폰트 깨짐 및 레이아웃 정렬 검증.
