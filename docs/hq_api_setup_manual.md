## 1. 토스페이먼츠 (POS 결제)

### 개요
- **목적:** 키오스크 및 매장 POS 카드/간편결제
- **비용 부담:** 각 매장 (수수료 약 2.5~3.5%)
- **소요 시간:** 가입 신청 ~ 승인까지 약 7~10 영업일
- **고객센터:** 1544-7772

### 매장 사전 준비물
```
✅ 사업자등록증
✅ 통장 사본 (정산받을 계좌)
✅ 대표자 신분증
✅ 사업장 주소
```

### 연결 절차 (매장 안내용)

#### Step 1: 토스페이먼츠 가맹점 가입
```
1. https://www.tosspayments.com 접속
2. [전자결제 신청하기] 클릭
3. 회원가입 → 사업자 정보 입력
4. 서류 제출 (사업자등록증, 통장사본, 신분증)
5. 입점 심사 + 카드사 심사 진행 (7~10일)
6. 심사 통과 → 계약 완료

💡 TIP: "미리 오픈" 서비스 이용 시 심사 중에도 
   신용/체크카드 일시불 결제를 먼저 받을 수 있음!
```

#### Step 2: API 키 확인
```
1. https://business.tosspayments.com (상점관리자) 로그인
2. 상점명 선택
3. [개발 연동] 메뉴 클릭
4. 아래 2가지 키를 확인:

   ┌────────────────────────────────────────┐
   │ 테스트 API 키 (개발/테스트용)           │
   │ - Client Key: test_ck_xxxxxxxx        │
   │ - Secret Key: test_sk_xxxxxxxx        │
   │                                        │
   │ 라이브 API 키 (실거래용) ← 이것을 사용! │
   │ - Client Key: live_ck_xxxxxxxx        │
   │ - Secret Key: live_sk_xxxxxxxx        │
   └────────────────────────────────────────┘
```

#### Step 3: Florasync 환경설정에 입력
```
1. Florasync 앱 → 환경설정 → 연동 및 자동화 → POS 결제
2. Client Key 붙여넣기
3. Secret Key 붙여넣기
4. [연결 테스트] 클릭 → ✅ 연결 성공!
5. 결제 수단 선택: 카드/간편결제/계좌이체 ON/OFF
```

### 본사 원격 지원 시 주의사항
```
⚠️ Secret Key는 절대 이메일/카톡에 일반 텍스트로 받지 않는다!
   → 전화 통화로 불러주거나, 화면 공유로 직접 입력하도록 안내
   → 또는 매장이 직접 앱에서 붙여넣기 (전화로 단계 안내)

⚠️ 반드시 "라이브 API 키"를 사용하도록 확인
   → "테스트 키"로 입력하면 실제 결제가 이루어지지 않음
```

### 트러블슈팅
| 증상 | 원인 | 해결 |
|------|------|------|
| 연결 테스트 실패 | 테스트 키 입력함 | 라이브 키로 변경 |
| 결제 시 오류 | 심사 미완료 | 토스 상점관리자에서 심사 상태 확인 |
| 특정 카드사 거절 | 해당 카드사 심사 미통과 | 토스 고객센터 문의 |
| 정산 안됨 | 계좌 정보 오류 | 상점관리자에서 정산계좌 확인 |

## 2. 카카오 알림톡

### 개요
- **목적:** 주문접수/제작완료/배달완료 등 자동 알림 발송
- **비용 부담:** 각 매장 (건당 약 8~15원)
- **대행사:** 솔라피(SOLAPI) 추천
- **소요 시간:** 채널 개설 1일, 템플릿 심사 1~3영업일

### 매장 사전 준비물
```
✅ 카카오톡 계정 (대표자 명의)
✅ 사업자등록증 (비즈니스 채널 인증용)
✅ 카카오톡 채널 프로필 이미지
```

### 연결 절차

#### Step 1: 카카오톡 채널 개설
```
1. https://center-pf.kakao.com (채널 관리자센터) 접속
2. 카카오 계정으로 로그인
3. [새 채널 만들기] 클릭
4. 채널명: "OO화원" (매장 이름)
5. 프로필 이미지 업로드
6. 채널 검색 허용: ON
7. 고객센터 정보 입력 (전화번호, 주소)
```

#### Step 2: 비즈니스 채널 인증
```
1. 채널 관리자센터 → [비즈니스 채널] 메뉴
2. [비즈니스 채널 신청] 클릭
3. 사업자등록증 업로드
4. 사업자 정보 입력
5. 심사 대기 (1~2 영업일)
6. ✅ 인증 완료 → 알림톡 발송 가능!

⚠️ "비즈니스 채널" 인증 없이는 알림톡 발송 불가!
```

#### Step 3: 솔라피 가입 및 연동
```
1. https://solapi.com 접속 → 회원가입
2. 로그인 → 대시보드
3. [카카오 채널 연동] 메뉴 클릭
4. 카카오톡 채널 검색용 아이디 입력
5. 관리자 휴대폰 번호로 인증
6. ✅ 연동 완료 → PFID (발신프로필 ID) 발급됨

7. [API 설정] 메뉴 → API Key 확인:
   ┌────────────────────────────────┐
   │ API Key: NXXXXXXXXXXXXXXXDX   │
   │ API Secret: XXXXXXXXXXXXXXXX  │
   └────────────────────────────────┘

8. 포인트 충전: [결제] → 충전하기 (최소 10,000원~)
```

#### Step 4: 알림톡 템플릿 등록
```
Florasync에서 사용할 템플릿 (본사가 기본 제공):

1. 주문 접수 알림
   "#{고객명}님, 주문이 접수되었습니다.
    상품: #{상품명} | 배송일: #{배송일}
    문의: #{매장전화}"

2. 제작 완료 알림 (완성 사진 첨부)
   "#{고객명}님의 꽃다발이 완성되었습니다! 🌹
    [사진 보기]"

3. 배송 출발 알림 (추적 링크)
   "#{고객명}님, 꽃이 배송 출발했습니다! 🚗
    [배송 추적하기]"

4. 배달 완료 알림 (배달 사진 첨부)
   "#{고객명}님, 꽃이 안전하게 전달되었습니다! 📸
    [배달 사진 보기]"

→ 솔라피에서 템플릿 등록 → 카카오 심사 (1~3영업일)
→ ✅ 승인 후 자동 발송 시작
```

#### Step 5: Florasync 환경설정에 입력
```
1. Florasync 앱 → 환경설정 → 연동 및 자동화 → 알림톡
2. 대행사: 솔라피 선택
3. API Key 붙여넣기
4. API Secret 붙여넣기
5. PFID (발신프로필 ID) 붙여넣기
6. [연결 테스트] → 테스트 알림 발송 → ✅ 성공!
7. 자동 발송 설정:
   - 주문 접수 시 [ON/OFF]
   - 제작 완료 시 [ON/OFF]
   - 배송 출발 시 [ON/OFF]
   - 배달 완료 시 [ON/OFF]
```

### 트러블슈팅
| 증상 | 원인 | 해결 |
|------|------|------|
| 알림 발송 안됨 | 포인트 소진 | 솔라피 포인트 충전 |
| 템플릿 심사 거절 | 변수 형식 오류 | 솔라피 가이드 참고하여 템플릿 재수정 |
| 채널 연동 실패 | 비즈니스 인증 미완료 | Step 2 비즈니스 채널 인증 먼저 완료 |
| PFID 오류 | 잘못된 채널 연동 | 솔라피에서 채널 재연동 |

## 3. 카카오 T 배송 (비즈퀵)

### 개요
- **목적:** 앱에서 바로 퀵/도보 배송 호출
- **비용 부담:** 각 매장 (배송건당 실비)
- **문의:** biz.cs@kakaomobility.com
- **소요 시간:** 상담 신청 ~ 키 발급까지 약 1~2주

### 매장 사전 준비물
```
✅ 사업자등록증
✅ 법인카드 또는 가상계좌 (결제 수단)
✅ 대표자 정보
```

### 연결 절차

#### Step 1: 카카오 T 비즈니스 가입
```
1. https://biz.kakaomobility.com 접속
2. 기업 회원 가입
3. 사업자 정보 입력
4. 결제 수단 등록 (법인카드 or 가상계좌)
5. 그룹 생성 → 직원 등록 (선택)
```

#### Step 2: API 연동 신청
```
1. biz.cs@kakaomobility.com 으로 이메일 발송:

   제목: 카카오 T 비즈퀵 API 연동 신청 - OO화원
   내용:
   - 사업자명: OO화원
   - 연동 목적: 꽃배달 퀵/도보 서비스
   - 연동 시스템: Florasync (화원 ERP+POS)
   - 예상 월 배송량: XX건
   - 담당자 연락처: 010-xxxx-xxxx

2. 카카오모빌리티 측에서 연동 상담 연락
3. 계약 체결 → 개발 가이드 + API 스펙 전달
```

#### Step 3: API 키 발급 및 테스트
```
1. 카카오 T 퀵∙도보 배송 API Sandbox 접속
2. 개발자 계정 로그인
3. Vendor ID + API Key 발급

   ┌────────────────────────────────┐
   │ Vendor ID: VND_XXXXXXXX       │
   │ API Key: XXXXXXXXXXXXXXXXXX   │
   └────────────────────────────────┘

4. 서버 IP 주소 등록 (Supabase Edge Function IP)
5. Sandbox 환경에서 테스트 호출
```

#### Step 4: Florasync 환경설정에 입력
```
1. Florasync 앱 → 환경설정 → 연동 및 자동화 → 배송
2. Vendor ID 입력
3. API Key 입력
4. [연결 테스트] → 테스트 배송 호출 → ✅ 성공!
5. 기본 설정:
   - 기본 배송 방식: 퀵 / 도보 선택
   - 발송지 주소: 매장 주소 자동 입력
```

### ⚠️ 본사 참고
```
카카오 T API는 개별 매장 단위 계약이 아닌,
플랫폼(Florasync) 단위 계약이 더 효율적일 수 있음.

→ 본사가 카카오모빌리티와 "플랫폼 파트너" 계약 체결
→ 각 매장은 Florasync를 통해 바로 이용
→ 정산은 각 매장별로 분리

→ 이 방식이 가능한지 카카오모빌리티 측에 확인 필요!
```

## 4. 네이버 스마트스토어

### 개요
- **목적:** 네이버 쇼핑 상품/주문/재고 자동 동기화
- **비용 부담:** 각 매장 (네이버 수수료)
- **소요 시간:** 이미 스마트스토어 운영 중이면 30분 내 연동 가능
- **필수 조건:** 스마트스토어센터 "통합 매니저" 권한 필요

### 매장 사전 준비물
```
✅ 네이버 스마트스토어 이미 운영 중이어야 함
✅ 통합 매니저 권한 계정
✅ 서버 공인 IP 등록 필요 (본사에서 안내)
```

### 연결 절차

#### Step 1: 커머스API센터 가입
```
1. https://apicenter.commerce.naver.com 접속
2. 스마트스토어 통합 매니저 계정으로 로그인
3. 커머스API센터 이용약관 동의
4. 가입 완료
```

#### Step 2: 애플리케이션 등록
```
1. 커머스API센터 → [애플리케이션 등록하기]
2. 입력 항목:
   - 애플리케이션 이름: "Florasync 연동"
   - 설명: "화원 ERP 재고/주문 연동"
3. API 호출 IP 등록 ← ⚠️ 매우 중요!
   → Florasync Supabase Edge Function의 공인 IP
   → 본사에서 IP 주소를 안내해야 함!
4. API 권한 설정:
   - ✅ 상품 API
   - ✅ 주문 판매자 API  
   - ✅ 판매자 정보 API
5. 저장 → 애플리케이션 생성 완료
```

#### Step 3: 인증키 확인
```
[내 애플리케이션] 메뉴에서 확인:

   ┌────────────────────────────────────────────┐
   │ 애플리케이션 ID: XXXXXXXXXXXXXXXXXXXXXXXXX │
   │ 애플리케이션 Secret: XXXXXXXXXXXXXXXXXXXXX  │
   └────────────────────────────────────────────┘
```

#### Step 4: Florasync 환경설정에 입력
```
1. Florasync 앱 → 환경설정 → 연동 및 자동화 → 온라인 마켓 → 네이버
2. 앱 ID 붙여넣기
3. 앱 Secret 붙여넣기
4. [연결 테스트] → 상품 목록 불러오기 → ✅ 성공!
5. 동기화 설정:
   - 상품 동기화 [ON/OFF]
   - 주문 자동 수신 [ON/OFF]
   - 재고 자동 동기화 [ON/OFF]
   - 가격 동기화 [ON/OFF]
```

### ⚠️ IP 등록 관련 본사 역할
```
네이버 API는 등록된 IP에서만 호출 가능!

→ 본사가 모든 매장에게 안내해야 할 IP:
   - Supabase Edge Function 공인 IP: XXX.XXX.XXX.XXX
   - 이 IP를 매장이 커머스API센터에 등록해야 함

→ IP가 변경되면 본사에서 전 매장에 공지하여 업데이트!
```

### 트러블슈팅
| 증상 | 원인 | 해결 |
|------|------|------|
| 연결 실패 | IP 미등록 | 커머스API센터에서 서버 IP 등록 |
| 권한 오류 | 통합 매니저 아님 | 스마트스토어센터에서 권한 변경 |
| 상품 안 보임 | API 권한 미선택 | 커머스API센터에서 상품 API 권한 추가 |
| 토큰 만료 | 인증 토큰 갱신 안됨 | 앱이 자동 갱신 (정상 동작 확인) |

## 5. 카페24

### 개요
- **목적:** 카페24 쇼핑몰 상품/주문/재고 자동 동기화
- **비용 부담:** 각 매장 (카페24 수수료)
- **소요 시간:** 약 30분~1시간

### 매장 사전 준비물
```
✅ 카페24 쇼핑몰 이미 운영 중이어야 함
✅ 카페24 통합 계정 (관리자)
```

### 연결 절차

```
⚠️ 본사가 카페24 개발자센터에 Florasync 앱을 미리 등록해야 함!

본사 사전 작업:

3. 앱 유형: Web Application
4. App URL: https://florasync.com/cafe24
5. Redirect URI: https://florasync.com/api/cafe24/callback
6. 저장 → Client ID + Client Secret 발급

→ 이것은 본사 1회 작업. 개별 매장은 "앱 설치"만 하면 됨!
```

#### Step 2: 매장에서 앱 설치
```
1. 카페24 쇼핑몰 관리자 로그인

```
1. Florasync 앱 → 환경설정 → 연동 및 자동화 → 온라인 마켓 → 카페24

## 6. 프린터 브릿지 (리본/폼택)

### 개요
- **목적:** 리본 프린터, 폼택 프린터, 메시지 카드 프린터 연결
- **비용 부담:** 없음 (하드웨어는 매장이 보유)
- **소요 시간:** 5분

### 연결 절차
```
1. 매장 PC에 프린터 브릿지 프로그램 설치
   → Florasync 앱 → 환경설정 → 인쇄 → [브릿지 다운로드]
   
2. 브릿지 설치 후 자동 실행
   → 포트: 8002 (기본)
   
3. 연결할 프린터 선택:
   → 리본 프린터: [프린터 목록에서 선택]
   → 폼택 프린터: [프린터 목록에서 선택]
   → 메시지 카드: [프린터 목록에서 선택]
   
4. [연결 테스트] → 각 프린터별 테스트 출력 → ✅ 성공!

5. 폼택 설정:
   → 용지 규격 선택: [6칸] [8칸] [12칸] [21칸]
   → 시작 위치: [자동] (뒤→앞 자동 이동)
```

### 트러블슈팅
| 증상 | 원인 | 해결 |
|------|------|------|
| 브릿지 연결 안됨 | 방화벽 차단 | 포트 8002 허용 설정 |
| 프린터 목록 비어있음 | 프린터 드라이버 미설치 | 프린터 제조사 드라이버 설치 |
| 인쇄 깨짐 | 용지 규격 불일치 | 폼택 규격 재선택 |
| 리본 위치 어긋남 | 리본 폭 설정 오류 | 환경설정에서 리본 폭 재설정 |

## 7. 원격 지원 절차

### 본사 슈퍼 어드민에서 원격 지원하기

```
1. 슈퍼 어드민 로그인
   → https://admin.florasync.com

2. [매장 관리] → 지원 대상 매장 검색

3. 매장 상세 → [연동 현황] 탭
   → 어떤 서비스가 미연결인지 확인

4. [원격 설정 지원] 클릭
   → 매장 환경설정 화면에 접근 가능
   → 매장 사장님의 동의 필요! (팝업 확인)

5. 사장님으로부터 API 키 전달받기:
   ⚠️ 안전한 전달 방법:
   ├── ✅ 전화 통화로 읽어주기 (추천)
   ├── ✅ 화면 공유로 사장님이 직접 입력
   ├── ⚠️ 카톡 전송 (보안 주의 - 전송 후 삭제 권유)
   └── ❌ 이메일 전송 (비추천)

6. 키 입력 → [연결 테스트] → 성공 확인

7. 지원 완료 → [원격 접근 종료] 클릭
```

### 원격 지원 체크리스트

```
□ 사장님에게 원격 접근 동의 받았는가?
□ API 키가 올바른 "라이브" 키인가? (테스트 키 아닌지 확인)
□ 연결 테스트를 성공했는가?
□ 자동 발송/동기화 설정을 확인했는가?
□ 사장님에게 정상 연결 확인 안내했는가?
□ 지원 후 원격 접근을 종료했는가?
```

## 8. 자주 묻는 질문 (FAQ)

### 일반

**Q: 모든 API를 다 연결해야 하나요?**
A: 아니요. 필요한 것만 ON하면 됩니다. AI 기능만 써도 충분히 유용합니다.

**Q: API 키가 뭔가요? 어렵지 않나요?**
A: 비밀번호 같은 것입니다. 각 서비스 사이트에서 발급받아서 Florasync에 붙여넣기만 하면 됩니다.

**Q: 비용이 이중으로 나가나요?**
A: 아닙니다. Florasync 구독료 + 사용한 외부 서비스 실비만 나갑니다.

### 토스페이먼츠

**Q: 심사가 얼마나 걸리나요?**
A: 보통 7~10영업일. "미리 오픈" 이용 시 심사 중에도 카드 결제 가능.

**Q: 수수료는 얼마인가요?**
A: 업종/매출에 따라 다르지만 보통 카드 2.5~3.5%. 토스페이먼츠 상담 시 확인.

### 카카오 알림톡

**Q: 건당 비용이 얼마인가요?**
A: 솔라피 기준 알림톡 건당 약 8.4원. 하루 20건 발송 시 월 약 5,000원.

**Q: 템플릿 심사에서 거절당했어요.**
A: 변수(#{...}) 형식이 맞는지 확인. 광고성 문구가 있으면 거절됨.
   → 본사에서 제공하는 기본 템플릿을 그대로 사용하면 거절 거의 없음.

### 네이버 스마트스토어

**Q: IP 등록을 어떻게 하나요?**
A: 본사에서 안내하는 서버 IP를 커머스API센터 → 내 애플리케이션 → IP 입력란에 추가.

**Q: 통합 매니저 권한이 없어요.**
A: 스마트스토어센터 → 직원관리에서 해당 계정을 통합 매니저로 변경.

### 프린터

**Q: 브릿지 프로그램이 자꾸 꺼져요.**
A: Windows 시작 프로그램에 등록되어 있는지 확인. 재설치 후 [시작시 자동 실행] 옵션 ON.
