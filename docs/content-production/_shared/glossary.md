# FloXync 콘텐츠 공통 용어집

문서·영상·쇼츠에서 표기를 통일합니다. **사장님용**과 **본사 CS용**을 구분합니다.

---

## 사장님용 ( `/docs/manual` · 영상 · 쇼츠 )

| 통일 표기 | 피할 표기 |
|-----------|-----------|
| FloXync | 플로싱크, Floxync |
| **환경 설정** | 환경설정(혼용 시 앱 UI와 동일하게) |
| **프린터/브릿지** (탭명) | Printer Setup |
| **영수증/라벨 브릿지 사용** | PP ON, POS bridge |
| **브릿지 다운로드** (버튼명) | /api/downloads/bridge |
| **인쇄 연결 프로그램** | ppbridge, 데몬, localhost |
| **연결됨** / **연결 안 됨** | online, heartbeat |
| **윈도우 앱** | Electron, desktop client |
| **안드로이드 앱** / **모바일 앱** | APK, Drive file id |
| **요금제 페이지** | pricing, /ko#pricing |
| **왼쪽 메뉴 하단 QR** | ANDROID_APK_DOWNLOAD_URL |

**설명 패턴:** `환경 설정 → ○○ 탭 → [버튼명]`

---

## 본사 CS용 ( `super_admin_operations_manual.md` )

내부 문서에만 사용 가능:

| 용어 | 설명 |
|------|------|
| PP 브릿지 | 영수증·라벨 인쇄 (웹용 ZIP) |
| RP / 리본 브릿지 | 리본 프린터 |
| `print_jobs` | 외부 접수 → 매장 PC 인쇄 큐 |
| `/api/downloads/windows-app` | GitHub releases |

사장님 전화 응대 시에는 **사장님용 표기**로 말합니다.

## 발주·이관 (세 가지 — 혼동 주의)

| 통일 표기 | 뜻 | 대상 |
|-----------|-----|------|
| **외부발주** | FloXync 미가입 도매·외주처에 맡기는 기록 | 거래처(공급업체) |
| **지점 이관** | 본사 조직(다매장) 안에서 다른 지점으로 주문 넘기기 | 같은 `organization` 소속 지점 |
| **회원사 수발주** | FloXync 가입 매장끼리 발주·수주 | 다른 tenant(회원사) |

**쓰지 않는 옛 용어:** 「협력사 수발주」(도매상·광고주 링크 구상) — **현재 기능 없음**. 매뉴얼·CS에서 사용 금지.

**앱 메뉴 표기:** 주문 더보기 → `외부 발주` · `지점 이관 요청` · `회원사 수발주 발주`

---
