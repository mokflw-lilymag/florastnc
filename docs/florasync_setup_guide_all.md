# 🌸 플로라싱크(FloraSync) 완벽 세팅 가이드

본 가이드는 플라워샵 운영의 모든 과정을 자동화하고 효율화하기 위한 **'플로라싱크 통합 관리 시스템'**의 세팅 방법을 상세히 안내합니다. 각 단계에 따라 차근차근 설정을 완료하시면, 사장님만의 강력한 **'디지털 컨트롤 타워'**가 완성됩니다.

## 1. 기본 주문 및 매출 정책
매장의 운영 방식에 최적화된 기본 데이터 산출 방식을 결정합니다.

*   **포인트 적립률**: 고객이 결제한 금액의 몇 %를 포인트로 적립할지 설정합니다. (예: 1%, 3%, 5%)
*   **할인율 옵션**: 상품 등록 시 자주 사용하는 할인율(%)을 미리 등록하여 빠르게 적용할 수 있습니다.
*   **수익 인식 기준**:
    *   **주문일 기준**: 주문이 들어온 시점에 매출로 집계 (당일 예약 위주 매장 권장)
    *   **결제 완료일 기준**: 실제 입금이 확인된 시점에 매출로 집계 (예약 주문이 많은 매장 권장)

## 2. 카카오 알림톡 자동 전송 (Solapi)
배송 사진과 주문 완료 메시지를 고객에게 자동으로 전송하기 위한 설정입니다.

### 🔑 API 정보 획득 방법
1.  [솔라피(solapi.com)](https://solapi.com)에 가입합니다.
2.  **연동 설정 > API Key 관리** 메뉴에서 새로운 키를 생성합니다.
3.  화면에 표시된 `API Key`와 `API Secret`을 복사하여 **'Key:Secret'** 형식으로 입력합니다.
    *   *예시: `CIDK...:SDFK...`*

### 📱 발신 채널 ID 설정
*   현재 사용 중인 **카카오톡 채널 아이디**를 `@아이디` 형식으로 입력합니다. (예: `@플로라싱크`)
*   솔라피 대시보드에서 '카카오톡 채널 연동'이 완료되어 있어야 실제 발송이 가능합니다.

## 3. 클라우드 사진 저장소 (Google Drive)
제작된 상품 사진을 안전하게 보관하고 고객에게 링크를 공유하기 위해 사용합니다.

### 📂 Folder ID 추출 방법
1.  구글 드라이브에 접속하여 사진을 저장할 **새 폴더**를 생성합니다.
2.  해당 폴더를 우클릭하여 **'공유' > '일반 액세스'를 '링크가 있는 모든 사용자'**로 변경합니다. (매우 중요!)
3.  브라우저 주소창(URL)의 맨 뒷부분에 있는 영문+숫자 조합이 **Folder ID**입니다.
    *   *주소 형식: `drive.google.com/drive_folders/1abcDEfg...`*
    *   이 중 `1abcDEfg...` 부분만 복사하여 입력합니다.

## 4. 데이터 자동 실시간 백업 (Google Sheets)
주문 내역과 지출 내역을 구글 엑셀 시트에 실시간으로 기록하여 재무 관리를 돕습니다.

### 📊 Spreadsheet ID 추출 방법
1.  구글 시트에서 새로운 시트를 만듭니다. (주문용, 지출용 각각 1개씩 권장)
2.  주소창의 `/d/`와 `/edit` 사이에 있는 긴 문자열이 **Spreadsheet ID**입니다.
    *   *주소 형식: `docs.google.com/spreadsheets/d/1X-Y...Z/edit`*
    *   이 중 `1X-Y...Z` 부분만 복사하여 입력합니다.

## 5. POS 연동 (이지체크/토스)
매장 포스기와 시스템을 연결하여 매출 데이터를 자동 동기화합니다.

*   **이지체크 (EasyCheck)**: 이지체크 관리자 페이지에서 제공하는 **Webhook URL**을 시스템에 등록해야 합니다. (문의: 본사 담당자)
*   **토스 POS (Toss)**: 토스 비즈니스 설정에서 발급받은 **API Key**를 입력합니다.
*   연동이 완료되면 **'연동 상태 대시보드'**에 초록색 램프가 켜집니다.

---

## 6. 배송 업무 자동화 (카카오 T / 리본브릿지)
배송 호출과 리본 프린트 업무를 통합 관리합니다.

*   **카카오 T 배송 키**: [카카오 T 비즈니스](https://biz.kakaomobility.com) 가입 후 개발자 센터에서 발급받은 API 키를 입력합니다.
*   **리본 브릿지(Bridge)**: PC에 리본브릿지 프로그램이 설치되어 있어야 하며, '연결됨' 상태일 때만 프린트가 가능합니다.

## 7. ✨ AI 홍보 마스터 (브랜드 DNA)
사장님의 매장을 24시간 홍보하는 AI 마케팅 비서를 설정합니다.

### 🎭 브랜드 페르소나 선택
*   **우아함 & 프리미엄**: 고급 꽃집 이미지를 강조하는 격식 있는 말투
*   **따뜻함 & 감성적**: 일상의 작은 행복을 전하는 다정한 말투
*   **트렌디 & 힙**: 최신 유행을 선도하는 젊은 감각의 말투

### 📣 홍보 주제 등록
*   사장님 매장만의 강점을 입력하세요. (예: "매일 아침 경매받은 싱싱한 꽃", "네덜란드 수입 튤립 전문")
*   AI는 이 주제를 바탕으로 날씨와 시즌에 맞춰 홍보 문구를 작성합니다.

### 🌐 SNS 계정 연동
*   인스타그램, 유튜브 등의 계정을 연결하면 AI가 작성한 글을 **사장님 대신 자동으로 업로드**합니다.
*   '자율 주행 모드(Auto-Pilot)'를 켜두면 매일 정해진 시간에 자동으로 홍보가 이루어집니다.
