# 안티그래비티 스킬: [결제 및 보안] "내 돈은 소중하니까"

## 사용 목적
결제 모듈(토스페이먼츠 등) 연동이나 Auth 세션 보안, 트랜잭션 흐름이 끊기지 않아야 하는 핵심 기능을 구현할 때 사용합니다.

## 시스템 프롬프트
"**@tosspayments/payment-sdk**를 이용해 결제 시스템을 구축 중이야. 결제 승인 전후로 **Supabase Auth** 세션을 체크하고, 결제 완료 후 서버 액션(Server Actions)을 통해 DB를 업데이트하는 전체 시퀀스를 설명해줘. 특히 결제 중 브라우저 이탈이나 네트워크 오류가 발생했을 때의 정합성 유지 전략을 알려줘."

## 핵심 가이드라인
* 결제 전후 Auth 상태에 대한 강제로직(Guards) 배치.
* Server Actions를 활용한 안전한 DB 트랜잭션.
* 네트워크 이탈, 브라우저 뒤로가기 상황에 대비한 Rollback 프로세스와 데이터 무결성 보장.
