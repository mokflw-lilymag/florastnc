# FloXync 콘텐츠 제작 허브

문서 매뉴얼 · 화면 녹화 영상 매뉴얼 · 상황극 쇼츠를 **한 곳에서 기획·제작·배포**하기 위한 작업 공간입니다.

---

## 폴더 구조

```
docs/content-production/
├── README.md                          ← 지금 보고 있는 총괄 계획서
├── 01-document-manual/
│   ├── PLAN.md                        ← 문서 매뉴얼 방법 + 목차(제목·내용)
│   └── drafts/                        ← AI/사장님 초안 (기능별 md·html)
├── 02-screen-video-manual/
│   ├── PLAN.md                        ← 녹화 방법 + 에피소드 목록
│   ├── scripts/                       ← 영상별 나레이션·클릭 스크립트
│   └── assets/                        ← 썸네일·챕터 이미지·원본 녹화 (git 제외 권장)
├── 03-situation-shorts/
│   ├── PLAN.md                        ← 쇼츠 컨셉·브랜드 톤
│   └── SCENARIOS.md                   ← 회차별 시나리오·대사·촬영 노트
└── _shared/
    ├── glossary.md                    ← 용어 통일 (리본/PP브릿지/픽업 등)
    └── release-checklist.md           ← 배포 전 검수 (앱 버전·메뉴명 일치)
```

> **Git에 넣지 않을 것**: `.mp4`, `.mov`, 대용량 PSD — `assets/`는 `.gitignore` 추가 권장

---

## 3가지 콘텐츠 역할 분담

| 채널 | 목적 | 독자 | 길이·형식 |
|------|------|------|-----------|
| **문서 매뉴얼** | 검색·교육·온보딩 | 신입·사장님 | HTML/MD, 기능 단위 |
| **영상 매뉴얼** | 눈으로 따라하기 | 급한 사장님·직원 | 3~12분, 화면 녹화 |
| **상황극 쇼츠** | 공감·유입·브랜드 | 잠재 고객·SNS | 30~60초, 연출 |

**원칙**: 같은 기능이라도 **메뉴명·버튼명·순서는 3채널 모두 동일** (`_shared/glossary.md` 기준).

## 역할 분담 — AI(에이전트) vs 사장님

### AI(에이전트)가 할 수 있는 것

| 작업 | 산출물 |
|------|--------|
| 기능별 문서 초안 (코드·화면 기준 사실 확인) | `01-document-manual/drafts/*.md` |
| 영상 나레이션·클릭 스크립트 | `02-screen-video-manual/scripts/EPxx.md` |
| 쇼츠 대본·컷 리스트·자막 초안 | `03-situation-shorts/SCENARIOS.md` 보강 |
| 기존 `/docs/manual` HTML 구조에 맞춘 이식 | `src/app/docs/manual` 연동 |
| Playwright 등으로 **데모용 스크린샷** 캡처 | `public/images/manual/` |
| 용어집·목차·제작 체크리스트 유지 | `_shared/*` |
| 앱 메뉴 변경 시 목차 diff 제안 | PR 코멘트 수준 |

### 사장님이 해야 할 것

| 작업 | 이유 |
|------|------|
| **실제 매장 화면으로 최종 검수** | 우리 앱은 국가·요금제·매장마다 메뉴가 다름 |
| **화면 녹화·음성 녹음** | 목소리·현장감·브랜드는 사람이 최고 |
| **쇼츠 출연·촬영·편집** | 연기·조명·BGM·업로드는 크리에이터 영역 |
| **유튜브·인스타·틱톡 채널 개설·업로드** | 계정·저작권·음원 라이선스 |
| **베타 매장 1곳 피드백 반영** | “이 표현 사장님 눈높이 맞나?” |
| **PP브릿지·프린터 실기 테스트** | 문서/영상에 넣을 실제 성공 화면 |

## 권장 제작 순서 (4주 스프린트 예시)

| 주차 | 문서 | 영상 | 쇼츠 |
|------|------|------|------|
| **1주** | 온보딩·주문·설정 초안 3편 | EP01 녹화·편집 | 파일럿 1편 촬영 |
| **2주** | 배달·고객·상품 3편 | EP02~03 | 시리즈 ①② 업로드 |
| **3주** | 리본·재고·경비 3편 | EP04~05 | 시리즈 ③④ |
| **4주** | 통합 목차·검색 | 플레이리스트 정리 | 월 4편 루틴 확정 |

**우선순위 (매장 사장님 필수)**  
1. 로그인·첫 설정·PP브릿지  
2. 주문 접수 (PC·모바일)  
3. 배달·픽업·인쇄  
4. 리본 프린터  
5. 재고·경비·리포트
