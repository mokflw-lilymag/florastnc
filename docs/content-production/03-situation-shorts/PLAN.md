# 3. 상황극 쇼츠 제작 계획

## 1) 방법 (How)

### 포지셔닝
**「꽃집 사장님의 하루 — FloXync 있으면 이렇게 달라진다」**  
교육 영상이 아니라 **공감 → 한 줄 해결 → 로고** 구조. 제품은 자연스럽게 노출.

### 포맷 스펙
| 항목 | 값 |
|------|-----|
| 길이 | 30~60초 (릴스·쇼츠·틱톡 공통) |
| 비율 | 9:16 세로 |
| 톤 | 코믹디·공감·약간 과장 OK, 비하 금지 |
| BGM | 저작권 프리 (YouTube Audio Library 등) |
| 자막 | 상단 고정, 2줄 이내, 키워드 색 강조 |
| CTA | 마지막 3초 — “FloXync 무료 체험” / QR |

### 시리즈 브랜딩
- **시리즈명**: `꽃집 사장님, 오늘도 버텼다` (가칭)
- **고정 캐릭터**: 사장님 1 + 알바 1 (또는 사장님 단독 1인 2역)
- **고정 엔딩**: “FloXync — Sync your Flower” + 1초 로고

### 제작 루프
```
시나리오(AI 초안) → 사장님 연출 수정 → 1일 촬영 3~5편 배치 → 편집 → 주 2~3회 업로드
```

### AI vs 사장님
| AI | 사장님 |
|----|--------|
| SCENARIOS.md 대본·컷 | 출연·촬영 |
| 후킹 문구 10개 제안 | 톤·밈 최종 선택 |
| 자막 텍스트 | 편집·업로드·댓글 |
