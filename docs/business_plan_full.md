# 1. Executive Summary

## 한 줄 정의

> **Floxync는 세계 최초의 AI 기반 화훼 산업 통합 플랫폼으로,
> 꽃집의 관리·원가·마케팅·매출을 하나의 앱으로 해결합니다.**

## 핵심 요약

| 항목 | 내용 |
|------|------|
| **문제** | 꽃집 사장님들은 매번 다른 레시피, 2~5일 유통기한, 정확한 원가 파악 불가, 마케팅 시간 부족 등 업종 특유의 문제에 시달리고 있으나, 이를 해결하는 전용 소프트웨어가 전 세계에 존재하지 않음 |

| **차별화** | 세계 최초/유일 기능 7개, 경쟁사 대비 독점 기능 14개 |
| **매출 효과** | 고객 매장 매출 +33% 상승 (월 +333만원) |

# 2. 문제 정의

## 2-1. 꽃집은 일반 소매점이 아니다

대부분의 POS/ERP 소프트웨어는 꽃집을 "일반 소매점"으로 취급합니다.
그러나 꽃집에는 다른 업종에 없는 **5가지 고유한 문제**가 있습니다.

### 문제 ①: 매번 다른 레시피

```
식당: "김치찌개" = 매번 같은 재료, 같은 양
꽃집: "5만원 꽃다발" = 매번 다른 꽃, 다른 양

→ 재고 추적이 불가능
→ 원가를 아는 꽃집 사장님 = 거의 0%
→ "30년 했는데 내가 얼마 벌고 있는지 몰라요"
```

### 문제 ②: 극단적으로 짧은 유통기한

```
일반 소매: 유통기한 수개월~수년
꽃집: 유통기한 2~5일!

→ 폐기율 15~30%
→ 매입 판단 실수 = 당일 손실
→ 한국 꽃집 월 평균 폐기 손실: 15~30만원
```

### 문제 ③: 마케팅 역량 부재

```
★ 소상공인 설문: "가장 어려운 것?" → 1위: 마케팅 (67%)

꽃집 사장님의 일과:
  05:00 - 새벽 도매시장
  07:00 - 매장 오픈 준비
  09:00~19:00 - 주문 제작 + 배달 + 고객 응대
  20:00 - 마감 정산
  
  "SNS 해야 하는 건 아는데... 시간이 없어요"
  "뭘 올려야 할지 모르겠어요"
  "마케팅 대행? 월 30만원은 부담돼요"
```

### 문제 ④: 필러플라워 계량 불가

```
안개꽃 1단 구매 → 꽃다발 5~10개에 나눠 사용
→ 정확히 얼마나 썼는지 측정 불가
→ 원가에서 빠지는 금액 = 월 수십만원
→ 이 문제를 해결한 소프트웨어 = 전 세계 0개
```

### 문제 ⑤: 수기 관리의 한계

```
한국 화원의 현실:
  80%+ 아직 수기 장부 또는 엑셀
  50%+ 재고 관리 자체를 하지 않음
  90%+ 정확한 원가를 모름
  95%+ 체계적 마케팅을 하지 않음

  → 범용 POS를 써도 "꽃집에 맞지 않아서" 결국 안 씀
```

## 2-2. 문제의 크기

```
💰 한국 내 꽃집 약 15,000개
💰 평균 월 매출 1,000~2,000만원
💰 평균 폐기 손실: 15~30만원/월
💰 마케팅 부재로 인한 매출 기회 손실: 추정 30%+
💰 원가 파악 불가로 인한 마진 누수: 추정 10~15%

→ 한국 화훼 소매 시장에서만 연간 수백억원의 비효율이 발생!
```

# 3. 솔루션

## 3-1. Floxync가 해결하는 방법

> **"꽃집의 현실을 아는 사람이, 현장의 진짜 문제를 AI로 해결한다"**

| 문제 | Floxync 솔루션 | 결과 |
|------|-----------------|------|
| 원가를 모른다 | AI 사진분석 + 필러 비율터치 + 레시피 관리 | **원가 정확도 0% → 85%** |
| 재고 추적 불가 | 자동 차감 + 분화 2단계 차감 | **일 30분 → 0분** |
| 마케팅 시간 없다 | SNS 7종 자동화 + 주간 캘린더 | **일 1시간 → 주 5분** |
| 폐기가 많다 | AI 유통기한 감지 → 플래시 세일 | **월 15만 → 5만원** |
| 고객 재방문 낮다 | 기념일 리마인더 + 자동 쿠폰 | **재구매율 +25~40%** |
| 비수기 매출 감소 | 정기구독 + 계절 캠페인 | **고정 수입 확보** |

## 3-2. 핵심 가치 제안 (Value Proposition)

```
기존 소프트웨어를 쓰면:
  → 시간이 절약된다 (관리가 편해진다)

Floxync를 쓰면:
  → 시간이 절약된다 (관리가 편해진다)
  → + 원가를 처음으로 안다 (마진이 보인다)
  → + 마케팅이 자동이다 (매출이 올라간다)
  → + 폐기가 줄어든다 (손실이 줄어든다)
  → + 고객이 다시 온다 (충성 고객이 된다)

  = "관리만 편해지는 앱은 많다. 매출이 올라가는 앱은 Floxync뿐!"
```

### ⑧ Anywhere Cloud Printing: 모바일-매장 릴레이 인쇄 (세계 유일)

집에서도, 배송 중에도 내 매장 프린터로 즉시 인쇄. 드라이버 설치가 필요 없는 클라우드 방식의 리본/카드 인쇄 시스템입니다.

- **방식:** 모바일 요청 → 클라우드 대기열 → 매장 Printer Bridge가 자동 수신 및 출력
- **장점:** 아이폰, 갤럭시, 아이패드 등 기기에 관계없이 전 세계 어디서든 매장 프린터 제어 가능
- **차별점:** 기존의 복잡한 드라이버 설정이나 와이파이 페어링 단계 전면 제거

촬영 한 번으로 재고 차감 + 원가 계산이 자동 처리됩니다.

- **기술:** Gemini Vision API
- **정확도:** 폼플라워 95%, 그린류 85%
- **사장님 추가 작업:** 0초

#### 하드웨어 호환성

| 프린터 | 용도 | 호환 상태 | 예상 비용 |
|--------|------|:---:|:---:|
| **Epson M105** | 영수증/라벨/폼택 (현재 꽃집 80%+ 사용) | ✅ 즉시 가능 | 이미 보유 |
| **Epson M 시리즈** | M105 외 모델 → 대부분 **리본 출력 용도**로 사용 중 | ✅ 즉시 가능 | 이미 보유 |

| **정원인더스트리 JW-1660** | 화훼 전용 감열 리본 프린터 (전용 소모품) | ✅ 호환 가능 | ~39만원 |
| **기타 프린터** | 고객이 원하는 기종 | 🔜 확장 예정 | - |

- **Epson M105 / M 시리즈:** 대한민국 꽃집에서 가장 많이 사용하는 프린터입니다. M105는 영수증·라벨·폼택 인쇄에 활용되며, M 시리즈의 다른 모델들은 이미 **리본 출력 용도로 현장에서 폭넓게 사용 중**입니다. 별도 구매 없이 기존 장비를 그대로 활용 가능합니다.

- **XPrint + 감열지 리본 (비용 절감 솔루션):** 기존에 택배 송장·라벨 인쇄 전용으로 사용하던 XPrint 기기를 Floxync의 Printer Bridge를 통해 **리본 인쇄 전용기로 전환**하여 사용할 수 있습니다. 꽃자재상에서 판매하는 **감열지 리본**을 사용하면 전용 소모품 대비 **소모품 비용을 크게 절감**할 수 있습니다.
  - ⚠️ **XPrint 유일한 제한:** 최대 출력 폭 **106mm**까지만 지원
  - ✅ **실무에서는 문제없음:** 일반 꽃집에서 100mm 이상 리본을 사용하는 경우는 **거의 없음** → 실질적 제약 아님

|-----------|:---:|:---:|

| **인쇄 방식** | 감열식 (전용 감열 리본) | 감열식 (자재상 감열지 리본) |
| **최대 출력 폭** | ~165mm | 다양한 폭 지원 |
| **인쇄 속도** | 130mm/sec (고속) | 일반 속도 |
| **소모품 (리본)** | 전용 소모품 → **비쌈** | 자재상 범용 감열지 → **저렴** |
| **소프트웨어** | 정원마스터 앱 (인쇄 전용) | **Floxync 통합** (ERP+AI+마케팅+인쇄) |
| **연동 기능** | 인쇄만 가능 | 주문 연동 → 자동 인쇄 |
| **모바일 지원** | Android 전용 앱 | **전 기기 지원** (iOS, Android, PC, Mac) |
| **기기 호환성** | PC/Android 제한 | **웹 기반 (기기 무관)** |
| **자동 컷팅** | ✅ 지원 | 수동 (기기에 따라) |

> **💰 핵심 비용 절감 포인트:**
> - 프린터 본체: 39만원 vs 10만원대 = **약 29만원 절감**
> - 소모품(리본): 정원인더스트리 전용 감열 리본은 자재상 감열지 리본 대비 **2~3배 비쌈**
> - 연간 소모품 비용 차이: 월 50건 출력 기준 **연 20~30만원 추가 절감**
> - 정원인더스트리는 "인쇄만" 하는 단독 솔루션, Floxync는 **ERP+AI+마케팅+인쇄 통합**

- **향후 확장:** 고객이 원하는 프린터 기종을 Printer Bridge에 드라이버로 추가하여 리본 출력에 사용할 수 있도록 지속 개발 예정입니다.

## 10-2. 왜 이 팀인가?

```
❌ "현장을 모르는 개발자만 있는 팀"
   → 꽃집의 진짜 문제를 모름 → 쓸모없는 기능만 만듦

❌ "개발을 모르는 사장님만 있는 팀"  
   → 아이디어는 있지만 구현 못함 → 외주에 의존

✅ "30년 현장 경험 + 기술 개발 능력" = Floxync
   → 현장의 진짜 문제를 알고, 직접 해결할 수 있는 유일한 팀
   → 필러플라워 비율 터치, 분화 2단계 차감 같은 기능은
     꽃집을 해본 사람만 설계할 수 있다!
```
