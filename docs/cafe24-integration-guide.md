# 🛒 플로라싱크 카페24 주문 자동 수집 연동 가이드

플로라싱크에 카페24 자사몰을 연동하면, 고객이 결제한 주문이 플로라싱크의 [주문 관리] 화면으로 실시간으로 들어옵니다. 
컴퓨터를 잘 모르시는 분들도 이 가이드만 순서대로 따라 하시면 5분 만에 완벽하게 연동할 수 있습니다!

---

## 🚀 1단계: 카페24 개발자센터에서 "열쇠(API 키)" 발급받기

카페24가 플로라싱크에게 **"이 쇼핑몰의 주문 정보를 가져가도 좋아!"**라고 허락해 주려면 특별한 열쇠(Client ID, Secret)가 필요합니다.

1. **카페24 개발자센터 접속하기**
   - [카페24 개발자센터(developer.cafe24.com)](https://developer.cafe24.com/)에 접속하여 쇼핑몰 관리자 아이디로 로그인합니다.

2. **새로운 앱 만들기**
   - 상단 메뉴에서 **[나의 앱]**을 클릭하고, 우측 상단의 **[앱 만들기]** 버튼을 누릅니다.
   - 앱 이름은 본인만 보는 것이니 `플로라싱크 연동` 정도로 적당히 지어주세요.

3. **권한 설정하기 (매우 중요 ⭐)**
   - 앱 설정 화면으로 들어가서 **[API 권한]** 탭을 클릭합니다.
   - 수많은 권한 목록 중에서 **`mall.read_order` (주문 읽기)** 권한을 찾아 체크해 줍니다. (이 권한이 있어야 주문 내역을 가져올 수 있습니다.)

4. **열쇠(키) 복사하기**
   - 앱 정보 창에 보면 영어와 숫자가 섞인 복잡한 코드가 2개 있습니다.
   - **`Client ID`** (클라이언트 아이디)
   - **`Client Secret`** (클라이언트 시크릿)
   - 이 두 가지를 복사해서 메모장에 잠깐 붙여넣어 두세요! (나중에 플로라싱크에 입력해야 합니다.)

---

## ⚙️ 2단계: 플로라싱크에 열쇠(키) 등록하기

이제 카페24에서 받아온 열쇠를 플로라싱크에 꽂아줄 차례입니다.

1. 플로라싱크 앱에 접속한 뒤, 좌측 메뉴에서 **[환경 설정] -> [쇼핑몰 연동 관리]**로 들어갑니다.
2. 화면에 보이는 **[카페24 개발자 API]** 스위치를 켜서 초록색으로 만듭니다.
3. 아래 입력 칸에 정보를 정확히 입력합니다:
   - **쇼핑몰 아이디 (Mall ID)**: 사장님의 카페24 쇼핑몰 관리자 아이디 (예: `lilymagflower`)
   - **Client ID**: 1단계에서 복사해 둔 Client ID 붙여넣기
   - **Client Secret**: 1단계에서 복사해 둔 Client Secret 붙여넣기
4. **[인증 테스트 및 저장]** 버튼을 누릅니다.
   - "설정이 저장되었습니다"라는 메시지가 뜨면 1차 성공입니다!

---

## 🔐 3단계: 쇼핑몰 로그인 (재인증) 하기

설정이 저장되었다면, 이제 카페24에 진짜로 로그인해서 문을 활짝 열어줄 차례입니다.

1. 열쇠 모양의 아이콘이 있는 초록색 **[카페24 로그인 연동] (또는 [재인증])** 버튼을 누릅니다.
2. 작은 팝업창이 뜨면서 카페24 로그인 화면이 나옵니다.
3. 사장님의 카페24 관리자 아이디와 비밀번호를 입력하고 로그인합니다.
4. 권한 허용 동의를 묻는 창이 나오면 **[동의함]**을 눌러줍니다.
5. "성공적으로 연동되었습니다" 메시지가 뜨면 모든 준비가 끝났습니다! 🎉

이제 **[주문 관리]** 페이지로 가서 **[쇼핑몰 동기화]** 버튼을 누르시면, 카페24의 주문들이 촤르륵 들어오는 마법을 보실 수 있습니다.

---

## ⏰ 알아두면 좋은 "14일의 마법 규칙" (토큰 갱신)

카페24는 고객의 개인정보를 아주 중요하게 생각합니다. 그래서 우리가 방금 발급받은 열쇠의 **유통기한을 딱 2시간**으로 제한하고 있습니다.

**"헉! 그럼 2시간마다 매번 로그인(재인증)을 다시 해야 하나요?"**
아닙니다! 걱정하지 마세요. 플로라싱크에는 **[자동 무한 갱신]** 기능이 탑재되어 있습니다.

### 💡 플로라싱크의 자동 갱신 원리
- 카페24는 2시간짜리 일반 열쇠를 줄 때, **14일 동안 쓸 수 있는 '스페어키(Refresh Token)'**를 하나 더 줍니다.
- 2시간이 지나 일반 열쇠가 썩어버리면, 플로라싱크 서버가 알아서 몰래 이 스페어키를 들이밀고 새 열쇠를 받아옵니다.
- **중요한 점:** 새 열쇠를 받아올 때, 카페24는 **새로운 14일짜리 스페어키도 덤으로 얹어줍니다.**
- 즉, 플로라싱크 앱을 쓰실 때마다 스페어키의 유통기한이 계속 **14일 뒤로 쭉쭉 밀려나는 것**입니다!

### 🚨 딱 한 가지 주의사항!
만약 사장님께서 해외여행을 가시거나 바쁘셔서 **플로라싱크를 14일(정확히는 336시간) 동안 단 한 번도 켜지 않으신다면**, 스페어키마저 썩어버리게 됩니다.

그렇게 되면 플로라싱크가 스스로 열쇠를 갱신할 수 없기 때문에, 동기화를 누르셔도 주문이 들어오지 않습니다.
이때는 당황하지 마시고, **[환경 설정] -> [쇼핑몰 연동 관리] 메뉴로 가셔서 초록색 [재인증] 버튼을 딱 한 번만 다시 눌러서 로그인해 주시면 됩니다.** 그러면 다시 14일 무한 갱신 모드가 시작됩니다!
