S0. 첫날 30분 — 로그인부터 테스트 주문까지

  1. floxync.com 접속 → 이메일·비밀번호로 로그인 (또는 회원가입)
  2. 왼쪽 맨 아래 환경 설정 → 상점 정보에 매장명·전화·주소 입력
  3. S3 배송비 (배달 하시면)
  4. S1 인쇄 연결 (프린터 쓰시면)
  5. 새 주문에서 테스트 주문 1건 저장 → 주문 목록에 보이는지 확인
  6. 사이드바 하단 📱 모바일 앱 QR로 폰 설치 (선택)

앱 받기: 부록

📱 모바일·PC

S6. 휴대폰으로 주문 넣기 (인쇄는 매장 PC)

폰에서 주문을 넣어도 영수증·주문서는 매장 PC 프린터에서 나옵니다. PC에 S1 인쇄 연결이 되어 있어야 합니다.

  1. 안드로이드 앱 설치 또는 모바일 브라우저로 로그인
  2. 왼쪽 새 주문 (또는 모바일 주문 화면)
  3. 고객·상품·배송/픽업 정보 입력 → 저장
  4. 매장 PC에서 주문서가 자동 인쇄되는지 확인

리본 인쇄는 모바일에서 안 됩니다 → S2

더 자세히: 새 주문

S7. 하루 마감 — 일일 마감 정산

이럴 때: 장 마감할 때 오늘 매출·현금 금고·미수금을 한번에 확인할 때

  1. 주문 목록 상단 → 일일 마감 정산 (또는 /dashboard/orders/daily-settlement)
  2. 날짜를 오늘로 맞춥니다
  3. 카드 네 장(총매출·수금·금고·주문 건수)을 확인합니다
  4. 금고 현금 상세에서 이월·현금 매출·지출·시재 입출금을 봅니다
  5. 맞으면 일일 정산 마감 버튼으로 저장

더 자세히: 일일 마감 정산 (상세)

S8. 문자·카톡 붙여넣기 — AI 주문

  1. 새 주문 화면으로 갑니다
  2. 상단 AI 주문 영역에 고객이 보낸 문자 전체를 붙여넣기
  3. 분석 또는 자동 채우기 실행
  4. 이름·주소·상품이 맞는지 눈으로 확인 후 수정
  5. 저장

더 자세히: 2-1 AI 원클릭 주문

S9. 단골 등록 · 포인트 · 기념일

  1. 왼쪽 고객 (CRM) 메뉴
  2. 고객 추가 또는 검색 후 선택
  3. 이름·전화·기념일·포인트·마케팅 동의 입력
  4. 주문할 때 고객 검색하면 포인트가 자동 반영됩니다 (5,000원 이상 등 규칙은 환경 설정 참고)

더 자세히: 고객 CRM · S10 매출 캘린더

S10. 매출 캘린더 — 기념일·자동 문자 (시작하기)

  1. 왼쪽 매출 캘린더 메뉴 (플랜에 따라 표시)
  2. 매출 캘린더에서 기념일 D-7·구매 후 탭과 Auto-Pilot 스위치 확인
  3. 설정에서 메시지 문구를 미리 저장 (실제 문자 발송 연동은 추후 예정)
  4. 고객 CRM에 기념일이 있어야 자동 대상이 잡힙니다 → S9

더 자세히: 3-2 매출 캘린더
