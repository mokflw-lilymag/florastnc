# 🎬 플록싱크(Floxync) 완벽 세팅 가이드 영상 스크립트

이 스크립트는 꽃집 사장님들이 영상을 보며 그대로 따라 할 수 있도록 **시각적 가이드(화면)**와 **음성 대본(나레이션)**을 1대1로 매칭하여 작성되었습니다. 모든 내용은 2024-2025년 최신 시스템을 기준으로 검증되었습니다.

---

## 📽️ 영상 개요
*   **제목**: 5분 만에 끝내는 플록싱크 '디지털 컨트롤 타워' 세팅법
*   **대상**: 플록싱크 SaaS 신규 가입 사장님
*   **준비물**: 컴퓨터, 구글 계정, 솔라피 계정, 카카오 비즈니스 채널

---

## [Scene 1: 인트로]
*   **화면**: 플록싱크 메인 대시보드가 부드럽게 줌인되며 등장. 세련된 비트의 배경음악.
*   **자막**: 사장님의 꽃집을 자동화하는 가장 빠른 방법
*   **나레이션**: "안녕하세요, 사장님! 플록싱크와 함께하게 된 것을 환영합니다. 오늘은 사장님의 매장을 자동화 머신으로 바꿔줄 '디지털 컨트롤 타워' 세팅법을 아주 쉽고 정확하게 알려드릴게요. 딱 5분만 집중해 주세요!"

---

## [Scene 2: 기초 공사 - 주문 및 매출 정책]
*   **화면**: [설정] 메뉴의 '주문 및 정책' 탭 화면.
*   **액션**: 마우스 커서가 '포인트 적립률'과 '수익 인식 기준'을 차례로 가리킴.
*   **나레이션**: "먼저 기초 공사부터 시작해볼까요? [설정] 메뉴에서 사장님 매장만의 규칙을 정해주세요. 단골을 만드는 포인트 적립률과, 매출을 언제 집계할지 결정하는 수익 인식 기준을 선택하고 [저장]을 눌러주세요. 이 설정이 모든 통계의 기초가 됩니다."

---

## [Scene 3: 카카오 알림톡의 핵심 - 솔라피(Solapi) 연동]
*   **화면**: 솔라피(solapi.com) 콘솔 화면과 앱의 알림톡 설정창을 반쪽씩 보여줌.
*   **액션**: 솔라피 [API Key 관리] 클릭 -> [+ 새 키 추가] 클릭하는 모습.
*   **나레이션**: "이제 가장 중요한 알림톡 연동입니다. 솔라피 사이트에 로그인한 뒤, 왼쪽 메뉴에서 [API Key 관리]를 눌러주세요. [+ 새 API Key 추가]를 누르면 긴 영문 숫자가 나옵니다. 자, 여기서 중요합니다! 화면에 나온 'API Key'와 'Secret'을 복사해서, 우리 앱의 입력창에 '콜론(:)'을 사이에 두고 넣어주세요. 예를 들어 '키값:비밀값' 이런 형식이 되겠죠?"
*   **자막**: 필수 입력 형식 -> API Key : API Secret

---

## [Scene 4: 알림톡 채널과 템플릿 코드]
*   **화면**: 솔라피의 [카카오톡 채널] 메뉴와 [템플릿 관리] 화면.
*   **액션**: 승인된 템플릿 옆의 '템플릿 코드'를 드래그하여 복사하는 모습.
*   **나레이션**: "키를 입력했다면, 이제 내 카카오 채널과 연결할 차례입니다. 솔라피 [카카오톡 채널] 메뉴에서 사장님의 채널을 연동하세요. 그리고 [템플릿 관리]에서 미리 승인받은 메시지의 '템플릿 코드'를 가져와 우리 앱에 넣어주세요. 이 코드가 있어야 고객님께 사진이 포함된 알림톡이 정확히 배송됩니다."

---

## [Scene 5: 구글 인프라 - 사진 저장과 데이터 백업]
*   **화면**: 구글 드라이브 폴더와 구글 시트의 주소창(URL).
*   **액션**: 주소창의 긴 ID 부분을 화살표로 강조하며 복사하는 동작.
*   **나레이션**: "사진 보관과 엑셀 백업도 아주 간단합니다. 구글 드라이브 폴더나 시트를 만들고 주소창을 보세요. 주소 중간에 있는 아주 길고 복잡한 영어 숫자들이 바로 '고유 ID'입니다. 이 부분만 쏙 복사해서 앱에 넣어주세요. 아참, 드라이브 폴더는 반드시 '링크가 있는 모든 사용자'로 공유 설정 하는 것, 잊지 마세요!"

---

## [Scene 6: POS 및 배송 인프라]
*   **화면**: 이지체크/토스 설정창과 카카오 T 비즈니스 샌드박스 화면.
*   **액션**: 카카오 T 비즈니스에서 'API 키 발급' 버튼을 누르는 모습.
*   **나레이션**: "매장 포스와 배송도 연결해볼까요? 이지체크나 토스에서 받은 API 정보를 입력하면 매출이 실시간으로 연동됩니다. 배송은 [카카오 T 비즈니스] 사이트의 [API 관리] 메뉴에서 발급받은 키를 넣어주세요. 이제 클릭 한 번으로 기사님 호출까지 끝납니다."

---

## [Scene 7: ✨ AI 홍보 마스터 - 브랜드 DNA]
*   **화면**: [AI 홍보 마스터] 메뉴의 페르소나 선택 카드들.
*   **액션**: '우아함 & 프리미엄' 카드를 클릭하고 홍보 주제를 입력하는 모습.
*   **나레이션**: "마지막으로 사장님의 24시 마케팅 비서, 'AI 홍보 마스터'입니다. 매장의 분위기에 맞는 페르소나를 선택하고, 사장님 매장만의 강점을 홍보 주제로 적어주세요. '자율 주행 모드'를 켜두면, AI가 알아서 글을 쓰고 인스타그램에 업로드까지 마쳐줍니다. 정말 놀라운 세상이죠?"

---

## [Scene 8: 마무리]
*   **화면**: 연동 대시보드의 모든 램프가 초록색으로 켜진 모습.
*   **자막**: 축하합니다! 모든 세팅이 완료되었습니다.
*   **나레이션**: "자, 대시보드에 초록불이 다 들어왔나요? 이제 사장님의 꽃집은 디지털 날개를 달았습니다. 세팅 중에 궁금한 점이 생기면 언제든 저희 1대1 문의를 이용해 주세요. 사장님의 성공을 플록싱크가 함께 응원합니다. 감사합니다!"

---

> [!IMPORTANT]
> **사장님을 위한 촬영 팁**:
> 1. 실제 사장님의 매장 화면을 녹화하실 때는 개인정보(실제 API Secret 등)가 노출되지 않도록 모자이크 처리에 유의해 주세요.
> 2. 목소리 녹음이 어려우시면 클로바더빙(CLOVA Dubbing) 같은 AI 성우 서비스를 이용하시면 훨씬 전문적인 느낌이 납니다.
