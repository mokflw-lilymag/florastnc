초기 세팅이 어려우신가요?

엑셀 업로드부터 디자인 템플릿까지 완벽한 맞춤형 세팅을 대행해 드립니다.

FloraSync 공식 매뉴얼

FloraSync는 온라인 주문부터 매장 출력, 향후 오프라인 POS까지 모든 꽃집 업무를 유기적으로 연결하는 차세대 SaaS 플랫폼입니다. 본 매뉴얼을 통해 완벽한 워라밸을 달성하세요.

[ 대시보드 전체 화면 캡처 자리 ]

1. 환경 설정 (기초 공사)

플로라싱크의 AI 기능과 원격 출력이 완벽하게 작동하려면 기초 데이터 세팅이 필수입니다. 이곳에서 등록한 카테고리와 템플릿은 전체 앱의 근간이 됩니다.

1-1. 상품 및 카테고리 관리

꽃다발, 바구니, 화환 등 매장의 카테고리를 설정하고 자주 판매하는 상품을 등록합니다. (엑셀 일괄 업로드 지원)

[ 상품/카테고리 설정 화면 캡처 자리 ]
  • 엑셀 업로드 버튼: 대량의 상품을 한 번에 등록할 수 있습니다.
  • 카테고리 연동: 여기서 설정한 카테고리는 주문 등록 시 AI가 상품명을 분석할 때 매칭 기준이 됩니다.

1-2. 디자인 템플릿 관리

리본, 카드, 인수증의 글꼴, 크기, 위치를 세팅합니다.

[ 디자인 템플릿 에디터 화면 캡처 자리 ]

💡 꿀팁: 한 번만 완벽하게 위치를 맞춰두면, 앞으로 주문이 들어올 때마다 이름과 경조사어만 바뀌어 자동으로 생성됩니다.

1-3. 쇼핑몰 연동 관리 (네이버/카페24)

스마트스토어와 카페24의 주문을 실시간으로 가져옵니다.

[ 쇼핑몰 API 연동 화면 캡처 자리 ]

※ 14일 무한 갱신 규칙

최초 1회 [재인증(로그인)] 버튼을 눌러 연동해두면, 앱을 사용할 때마다 자동으로 토큰이 14일씩 갱신됩니다. 즉, 14일 넘게 아예 접속을 안 하지 않는 이상 두 번 다시 로그인하실 필요가 없습니다.

2. 주문 관리 & AI 주문 입력

더 이상 손으로 타이핑하지 마세요. 쇼핑몰 주문은 자동으로 들어오고, 카톡/문자 주문은 AI가 해결합니다.

💬 AI 원클릭 주문 입력

[ AI 주문 텍스트 붙여넣기 창 캡처 ]

고객이 보낸 카톡("내일 오후 2시까지 여의도 자이아파트로 10만원짜리 꽃바구니 보내주세요. 리본은 축하합니다/홍길동")을 그대로 복사해서 붙여넣으세요.

AI가 수령인, 날짜, 주소, 상품명, 리본 문구 좌/우를 완벽하게 분리하여 칸에 채워 넣습니다.

🛒 다채널 통합 관리

[ 쇼핑몰 뱃지가 달린 주문 리스트 캡처 ]

스마트스토어(N), 카페24(C), 카카오톡(K) 등 모든 채널의 주문이 하나의 리스트로 통합됩니다. 필터 버튼을 통해 채널별, 날짜별로 한눈에 확인하세요.

3. 원격 클라우드 출력 Killer Feature

타사 서비스에는 없는 플로라싱크만의 독보적 기능! "안방에서 누르면 매장에서 나옵니다."

🖨️ 어떻게 작동하나요?

[ 모바일 화면의 출력버튼 + 매장 프린터 분할 캡처 ]
  • 1

    퇴근 후 집이나 외부에서 스마트폰/패드로 플로라싱크 주문 상세창을 엽니다.

  • 2

    환경설정에서 미리 만들어둔 템플릿이 적용된 주문건의 [리본 출력], [카드 출력], [인수증 출력] 버튼을 누릅니다.

  • 3

    명령이 클라우드를 타고 매장에 켜져있는 PC로 즉시 전송되며, 매장 프린터에서 리본과 영수증이 쫙쫙 출력됩니다!

5. 넥스트 스텝: 오프라인 POS 연동 (예정)

[ POS 연동 개념도 일러스트 ]

온/오프라인 매출의 완벽한 통합

현재 입력하고 계신 기초 자료(카테고리, 상품)와 온라인 주문 데이터는 결코 버려지지 않습니다. 추후 업데이트될 플로라싱크 POS와 실시간으로 연동되어, 워크인(Walk-in) 고객의 현장 결제 데이터까지 이 화면 하나에서 통합 관리하게 될 것입니다.

꽃집 비즈니스의 완벽한 스케일업, 플로라싱크가 함께합니다.
