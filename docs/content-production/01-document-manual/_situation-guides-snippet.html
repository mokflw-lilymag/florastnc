🖥️ 매장 PC 📱 폰은 주문만 ⏱️ 약 15분

S1. 영수증·주문서 인쇄하기 (휴대폰 주문도 매장에서 출력)

이럴 때: 웹(크롬)만 쓰는데 프린터가 안 나오거나, 폰으로 받은 주문도 매장에서 자동으로 뽑고 싶을 때

끝나면 이렇게 됩니다

  • 환경 설정에 연결됨 표시
  • 주문 저장 시 영수증·주문서가 매장 프린터에서 나옴
  • 휴대폰·다른 PC에서 넣은 주문도 같은 PC에서 인쇄

순서 (윈도우 앱 쓰시면 부록 보시고 이 절은 건너뛰세요)

  1. 왼쪽 맨 아래 환경 설정 클릭
  2. 왼쪽 탭 프린터/브릿지 선택
  3. 영수증/라벨 브릿지 사용 스위치를 켭니다
  4. 브릿지 다운로드 → ZIP 풀기 → install.bat 관리자 권한 실행
  5. 상태 체크 → 연결됨 확인
  6. 프린트 테스트 → 종이 한 장 나오는지 확인
  7. 아래 프린터 기기 선택에서 영수증 프린터 고르기

더 자세히: 1-5 프린터/브릿지

🖥️ PC에서만 📱 모바일 인쇄 불가

S2. 리본 프린터 설정하기

📱 안드로이드 앱·휴대폰 웹에서는 리본을 인쇄할 수 없습니다. 반드시 매장 PC에서 해 주세요.
  1. 환경 설정 → 프린터/브릿지 → 리본 프린터 브릿지 사용 켜기
  2. 왼쪽 메뉴 리본 프린터 (제작·출력 그룹) 클릭
  3. 축하 문구 입력 → 미리보기 확인 → PRINT

더 자세히: 리본 프린터 (상세)

🖥️ 설정은 PC ⏱️ 약 10분

S3. 배송비 규칙 설정하기

한 번만 해 두면, 새 주문할 때 주소만 넣어도 배송비가 자동으로 들어갑니다.

  1. 환경 설정 → 탭 배송
  2. 기본 배송비 — 목록에 없는 지역일 때 쓸 금액
  3. 무료 배송 기준 — 이 금액 이상 주문이면 배송비 0원 (안 쓰면 0)
  4. 지역별 배송비 — 구·동 이름 + 금액 추가 (예: 강남구 5,000원)
  5. 화면 아래 저장 (있으면) 또는 입력 시 자동 저장 확인

주문할 때 배송비가 이상하면: 새 주문 화면에서 배송비 수동 스위치로 직접 수정할 수도 있어요.

더 자세히: 1-3 배송비 설정

🖥️·태블릿 ⏱️ 1~2분/건

S4. 빠른판매 POS — 매장에서 바로 결제

전화 주문 없이, 손님이 매장에 와서 바로 사 가실 때. 배송·실배송비 단계는 없습니다.

어디서 열기

  • 모바일 주문 화면 상단 ⚡ 빠른판매 POS 링크
  • 또는 주소창에 로그인 후 /dashboard/pos/quick
  1. 상품 카드 탭해서 담기 (또는 금액 직접 입력)
  2. 할인 필요하면 선택
  3. 결제 수단 — 카드·현금·계좌이체 등 탭
  4. ✓ 결제 완료 버튼

끝나면

주문 완료 + 결제 결제완료 — 매장 수령(store) 주문으로 기록됩니다.

🖥️ 권장 ⏱️ 주문 1건당 5분

S5. 픽업·배송 마무리 (결제·실배송비·지출까지)

이럴 때: 배송 나갔는데, 택배비 지출·결제 정리·완료 처리를 한곳에서 끝내고 싶을 때

단계어디서무엇을시스템에 남는 것
①새 주문배송지·배송비·리본·결제 수단·결제완료/미수 선택 후 저장주문 생성
②배송 (왼쪽 메뉴)오늘 배송 목록에서 해당 주문 찾기—
③같은 화면실제 배송비 입력 (카드·현금 나눠 넣기 가능) 후 저장지출에 배송비 자동 기록
④같은 화면배송 완료 처리배송완료
⑤(선택) 일일 마감 정산오늘 결제·미수·배송비 확인하루 정리

픽업만 하는 주문은 ③ 실제 배송비를 건너뛰어도 됩니다.

더 자세히: 새 주문 · 배송·픽업 · 일일 마감 · 지출

🖥️ PC 권장 ⏱️ 약 3분

외주처에 맡길 때 — 외부발주

이럴 때: FloXync를 쓰지 않는 도매·외주처에 맡기고 발주 기록·지출만 남길 때

  1. 거래처에 외주처 등록
  2. 주문 목록 → 더보기(⋮) → 외부 발주
  3. 외부발주처·단가 입력 → 인수증 복사 → 카톡 전달
  4. 외부발주주문 저장

더 자세히: 외부발주 (메뉴 백과)
