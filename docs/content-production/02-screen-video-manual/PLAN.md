# 2. 화면 녹화 영상 매뉴얼 제작 계획

## 1) 방법 (How)

### 콘셉트
**「3분 만에 따라하기」** — 문서보다 느리지만, 멈추며 따라 할 수 있는 속도.  
나레이션: **존댓말 + 짧은 문장** (“지금 보시는 것처럼, [저장]을 눌러 주세요.”)

### 녹화 환경
| 항목 | 권장 |
|------|------|
| 해상도 | 1920×1080 (최소 1280×720) |
| 도구 | OBS Studio / Camtasia / Windows Xbox Game Bar |
| 브라우저 | Chrome, 줌 100%, 북마크바 숨김 |
| 계정 | **데모 매장** 전용 (실고객 데이터 X) |
| 마이크 | 핀마이크 또는 조용한 방, -14 LUFS 목표 |
| 커서 | 크기 Large, 클릭 하이라이트 ON |

### 편집 규칙
- 인트로 5초: 로고 + 에피소드 제목
- 챕터 카드: 단계 전환 시 1초 정지 + 텍스트 (“2. 배달 정보 입력”)
- 실수·로딩 컷: dev 컴파일 대기는 편집에서 삭제
- 아웃트로 5초: “다음 편 — 주문 목록” + QR/매뉴얼 링크
- 자막: 한국어 burn-in 또는 YouTube 자동 + 수정

### 배포
- **YouTube** 플레이리스트: `FloXync 매장 매뉴얼`
- 썸네일 템플릿: 보라+에메랄드, 큰 한글 제목
- 문서 매뉴얼 각 장 하단에 해당 EP 링크 embed

### AI 협업
- `scripts/EPxx.md`에 **초 단위 스크립트** 작성 → 사장님 읽으며 녹화
- 녹화 후 AI가 자막 SRT 초안 생성 (Whisper 등, 사장님 업로드)

## 3) 샘플 스크립트 구조 (`scripts/EP02.md` 예시 골격)

```markdown
# EP02 PP 브릿지 설치 (8:00)

| 시간 | 화면 | 나레이션 |
|------|------|----------|
| 0:00 | 인트로 카드 | "오늘은 영수증 자동 인쇄, PP 브릿지 설치입니다." |
| 0:15 | 설정 → 프린터 | "왼쪽 [설정], 아래로 내려 [영수증/라벨 브릿지]입니다." |
| 0:45 | 브릿지 다운로드 | "[브릿지 다운로드]를 누르면 ZIP이 받아집니다." |
| ... | install.bat | "압축을 푼 뒤 install.bat을 관리자 권한으로 실행합니다." |
```
